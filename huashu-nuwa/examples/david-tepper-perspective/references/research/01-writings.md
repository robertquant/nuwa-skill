### 1. 2008-2009金融危机期间银行股抄底

**来源**: 多家媒体综合报道、华尔街日报引用  
**可信度**: ⭐⭐⭐⭐ 高（多源交叉验证）

**案例详情**:

| 要素 | 内容 |
|------|------|
| **时间** | 2009年初 |
| **投资标的** | 花旗(Citigroup)、美国银行(Bank of America)优先股 |
| **投资逻辑** | 判断美国政府不会让主要银行倒闭 |
| **买入时机** | 市场极度恐慌，银行股跌至历史低点 |
| **结果** | 基金2009年回报132%，获利约70亿美元 |
| **个人收益** | 泰珀当年个人赚约40亿美元 |

- 泰珀判断：政府会"眨眼"（妥协），不会让银行完全国有化

## 投资方法论（从访谈中提炼）

### 核心原则

| 原则 | 说明 | 来源 |
|------|------|------|
| **逆向投资** | 在市场恐慌时买入，在市场亢奋时卖出 | 多次访谈 |
| **困境资产专精** | 专注于破产边缘的公司债券和股票 | 传记资料 |
| **政策解读能力** | 准确判断央行和政府的干预边界 | 2009案例 |
| **仓位集中** | 高确信度时敢于重仓 | 13F持仓特点 |
| **等待耐心** | 大部分时间等待，关键时刻出击 | 经典语录 |

### 决策框架（推断）

根据其言行，推测泰珀的决策流程：

1. **宏观环境判断** → 解读央行/政府政策倾向
2. **资产筛选** → 找到极度低估的困境资产
3. **底线思维** → 最坏情况下能承受什么损失
4. **时机选择** → 等待市场恐慌情绪达到顶点
5. **重仓执行** → 在确信度高时大胆建仓
