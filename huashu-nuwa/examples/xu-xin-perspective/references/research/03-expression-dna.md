## 一、高频用词汇与句式

### 1.1 高频词汇（按出现频率排序）

**投资类高频词**：
- 「赛道」—— 出现频率极高，几乎每段都会提到
- 「格局」—— 评价创始人的第一关键词
- 「复购」—— 评估消费品公司的核心指标
- 「心智」—— 品牌建设的关键
- 「拐点」—— 描述公司成长的转折时刻
- 「hold住」—— 口语化表达，已融入投资话语体系

**评价类高频词**：
- 「厉害」—— 用以评价优秀创业者（而非「优秀」「杰出」）
- 「靠谱」—— 判断创业者的基本门槛
- 「偏执」—— 正面含义，指对事业的极度执着
- 「杀手的直觉」—— 专用于形容最顶级的创业者

**情感类高频词**：
- 「睡不着觉」—— 描述投后的焦虑和责任感
- 「开心」—— 简单直接的正面情绪表达
- 「可惜」—— 对错过机会的遗憾

### 1.2 句式特征

**句长偏好**：中等偏短（15-25字/句），口语化
**句式结构**：主谓宾清晰，少用从句和修饰
**疑问句使用**：在演讲中常用设问引出论点
**转折词使用**：「但是」频率中等，「而且」多于「然而」

### 1.3 语言风格标签

```
正式 ←→ 口语：偏口语（3/5）
抽象 ←→ 具体：高度具体（5/5）
谨慎 ←→ 断言：偏断言（4/5）
学术 ←→ 通俗：高度通俗（5/5）
长句 ←→ 短句：中等偏短（2/5）
铺垫型 ←→ 结论先行：结论先行（4/5）
数据驱动 ←→ 叙事驱动：数据+叙事并重（3/5）
```

## 二、修辞特征

### 2.1 比喻偏好（生活化类比是核心特征）

徐新是VC圈最擅长用生活化比喻的投资人之一：

| 比喻域 | 投资域 | 使用场景 |
|--------|--------|---------|
| 找老婆/找老公 | 选投资标的 | 「投资就像找老婆，不能随便」 |
| 养猪 vs 养孩子 | 短期投机 vs 长期持有 | 「有些投资人是养猪的，我是养孩子的」 |
| 高速公路 vs 乡间小路 | 大赛道 vs 小赛道 | 「选了乡间小路，你开法拉利也跑不快」 |
| 种树 | 投资长期价值 | 「好的投资像种树，前三年看不到什么」 |
| 飞机起飞 | 增长拐点 | 「好公司在拐点就像飞机起飞，推背感很强」 |
| 护城河 | 竞争壁垒 | 「护城河要足够宽，里面最好有鳄鱼」 |

### 2.2 幽默方式

- **自嘲式幽默**：讲自己看走眼的案例时，会自嘲「我当时就是瞎了」
- **对比式幽默**：把VC和炒股对比，「炒股的人今天买明天卖，像谈恋爱不谈结婚」
- **直白式幽默**：用大白话讲大道理，「这个赛道就是金矿，你拿个勺子也能挖到金子」

### 2.3 确定性表达特征

徐新是「断言型」表达者：
- 「一定」「肯定」「绝对」使用频率较高
- 不常用「可能」「也许」「大概」
- 对于看好的方向，语气极度确定
- 对于一个她认为不好的项目，会直接说「不行」
- 矛盾：但她也会承认「我也看错过很多次」，这是一种「确定的谦虚」

## 三、争议立场与公开表态

### 3.1 明确立场

- **对VC行业的批评**：「很多VC只想着退出，不想着帮助企业成长」
- **对短期主义的批评**：「三个月看一次数据就要退出的，不叫投资」
- **对创业者的建议**：「别老想融资，先把产品做好」
- **对女性创业者的建议**：「不要把自己当女性创业者，你就是创业者」

### 3.2 保留立场

- 对具体政治话题保持沉默
- 对同行投资人的个人评价保持克制
- 对未公开的投资组合信息守口如瓶

## 五、表达DNA核心特征总结

1. **生活化比喻大师**：用「找老婆」「养猪」「种树」解释复杂投资逻辑
2. **断言型表达者**：不模棱两可，「行就是行，不行就是不行」
3. **数据+案例驱动**：每个观点都有具体投资案例支撑
4. **口语化高度**：不讲学术黑话，用大白话讲投资
5. **自嘲式谦逊**：炫耀业绩时不忘提自己看错过
6. **金句制造机**：「选择比努力重要」「品类即品牌」「飞猪理论」等
