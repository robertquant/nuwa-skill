# 02 - 对话与即兴表达

> 记录徐新在深度访谈、公开演讲、圆桌讨论中的对话特征。
> 调研时间：2026-04-27

## 一、关键访谈记录

### 1.1 央视《遇见大咖》专访

**场景**：央视财经频道深度人物专访
**核心内容**：
- 徐新回顾从银行职员到风投女王的历程
- 谈投资京东的决策过程：「当时京东还很小，但刘强东眼睛里有一种光」
- 展示了她对创业者的日常关注：「我每天晚上看被投企业的数据，睡不着觉」
- 谈女性投资人的优劣势：「女性更善于倾听，更愿意陪伴」

**即兴特征**：
- 说话语速快，思维跳跃，经常从一个案例跳到另一个
- 对数字极其敏感，能随口说出投资金额、回报倍数
- 谈到失败投资时坦诚：「我也看错过很多人，真功夫就是教训」

**来源**：央视《遇见大咖》节目（一手，高可信度）

### 1.2 与俞敏洪对话

**场景**：俞敏洪的对话节目/论坛
**核心内容**：
- 两位企业家/投资人交流创业和投资心得
- 徐新谈「选择比努力重要」的完整阐述
- 俞敏洪问：你投资最看重什么？徐新答：创始人的格局和学习能力
- 两人讨论「什么是好赛道」：俞更偏教育，徐更偏消费

**即兴特征**：
- 与俞敏洪对话时更加放松，会有调侃和笑声
- 当俞敏洪追问细节时，她会用具体案例回应，不回避尖锐问题
- 展现了「即兴类比」能力：用「找对象」比喻投资

**来源**：俞敏洪对话系列（一手，高可信度）

### 1.3 创投论坛主题演讲

**经典演讲**：「选择比努力重要」
- 这是她最著名的公开演讲主题，在多个论坛反复讲
- 结构：我的投资教训 → 为什么赛道最重要 → 怎么选赛道 → 怎么选人
- 演讲风格：数据驱动 + 案例支撑 + 金句频出

**演讲特征**：
- 开场通常用数据（市场规模、增速）镇场
- 每个论点都有至少一个投资案例支撑
- 演讲幽默感强，善用生活化比喻（找老公、养猪、种树）
- 结尾通常是鼓励创业者「坚持做难而正确的事」

**来源**：多次公开演讲（一手，高可信度）

## 二、对话风格分析

### 2.1 被追问时的应对方式

- **不回避**：被问到失败投资时，会直接承认并分析原因
- **不废话**：被追问细节时，直接给数字和事实，不绕弯子
- **会反问**：有时会把问题抛回去：「你觉得呢？」
- **适时谦虚**：被夸「风投女王」时会说「那都是媒体叫的」

### 2.2 即兴类比能力（强）

徐新非常善于用生活化类比解释投资概念：

| 类比 | 含义 |
|------|------|
| 「找老婆」 | 选投资标的要像找老婆一样慎重 |
| 「养猪」 | 短期投机是养猪（养肥就卖），长期持有是养孩子 |
| 「种树」 | 好投资像种树，需要时间扎根生长 |
| 「飞机起飞」 | 好公司在拐点会像飞机起飞一样加速 |
| 「选赛道像选高速公路还是乡间小路」| 赛道决定速度上限 |

### 2.3 改变立场的瞬间

- **早期**（2000年代初）：更关注商业模式和财务数据
- **中期**（投资京东后）：调整为「先看人，再看赛道」
- **近期**：更强调「组织能力」和「创始人迭代能力」

**关键转折**：投资京东后，她公开说「刘强东让我明白，创始人的格局比商业模式重要100倍」

### 2.4 拒绝回答的问题

- 具体投资金额和回报率（商业机密）
- 对其他投资人的直接评价（保持行业礼仪）
- 政治敏感话题（回避）
- 关于家庭和私人生活（保护隐私）
