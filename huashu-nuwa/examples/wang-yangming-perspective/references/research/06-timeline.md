## 一、家族背景与少年时期（1472-1490）

| 时间 | 年龄 | 事件 | 思想意义/影响 |
|------|------|------|--------------|
| 1472年（成化八年） | — | 九月三十日亥时，出生于浙江绍兴府余姚县龙泉山瑞云楼。母怀胎十四月，祖母岑氏梦神仙驾祥云送子，祖父竹轩翁取名"云" | 出生传说预示不凡，后世尊为"心圣" |
| 1476年（成化十二年） | 5岁 | 仍不语。有神僧过家门曰："好个孩儿，可惜道破"。祖父改其名为"守仁"（取《论语》"知及之，仁不能守之，虽得之，必失之"），遂开口说话 | 名字之变暗示"守仁"志向，"道破"传说反映神秘气质 |
| 1481年（成化十七年） | 10岁 | 父亲王华中状元，授翰林修撰，全家迁居京师。王氏为千年文化世家，远祖可追溯至东晋王羲之，但至王伦一代已淡泊功名，以教书为业 | 状元之子身份奠定其早年的功名期待与文化熏陶 |
| 1482年（成化十八年） | 11岁 | 随祖父竹轩公王伦赴京师投奔父亲。途中过金山寺，即兴赋诗，显露诗才 | 竹轩公潜移默化的文化启蒙，是其成长的根基 |
| 1484年（成化二十年） | 13岁 | 母亲郑氏去世，甚悲恸。祖父竹轩公与父亲王华成为其最亲密的家人 |母亲的离世促其早熟，对生死有初步体悟 |
| 1486年（成化二十二年） | 15岁 | 出游居庸三关，习骑射，甚至击败胡人。尝言："若朝廷用我，当提数万兵，讨平边疆"，有经略四方之志 |少年豪气不羁，军事志向初现，为日后军事才能埋下伏笔 |
| 1488年（弘治元年） | 17岁 | 回余姚成婚于南昌，娶诸氏。新婚之日入铁柱宫与道士谈养生，竟忘归 | 对道家长生术的兴趣，反映其对生命终极问题的探索 |
| 1489年（弘治二年） | 18岁 | 偕夫人归余姚，识大儒娄谅，信"圣人必可学而致之"。一改活泼性格，严肃求成圣人 | **【关键转折：立志】** 始有圣人之志，为一生思想追求奠定方向。同年"格竹"失败，朱子格物之学遭质疑 |
| 1490年（弘治三年） | 19岁 | 祖父竹轩公王伦去世。王伦终生未仕，淡泊名利，如陶渊明之风，对阳明人格影响深远 | 祖父的隐逸气质与孝悌家风构成其人格底色 |

## 二、科举与仕途早期（1490-1506）

| 时间 | 年龄 | 事件 | 思想意义/影响 |
|------|------|------|--------------|
| 1492年（弘治五年） | 21岁 | 参加浙江乡试中举。始实践朱子格物之学，"格竹"七日致病，质疑外求之理 | "格竹"失败是其思想困境的起点，为龙场悟道埋下伏笔 |
| 1493年（弘治六年） | 22岁 | 会试下第。归余姚，结龙泉诗社，对弈联诗 | 科举挫折中的文学休闲，展现其多面才华 |
| 1497年（弘治十年） | 26岁 | 寓京师，苦学诸家兵法，想借"雄"成圣 | 军事才能的培养与"借雄成圣"的尝试，反映其事功与思想结合的追求 |
| 1499年（弘治十二年） | 28岁 | 中进士，二甲第七。观政工部。受命督造威宁伯王越墓，以兵法部署工人 | 初显军事组织才能。同年，溺于养生术，有出尘之意 |
| 1500年（弘治十三年） | 29岁 | 授刑部主事。开始接触佛教，游九华山，与僧道论学 | 出入佛老，思想探索期。对禅宗的体悟或影响日后心学的直观倾向 |
| 1502年（弘治十五年） | 31岁 | 告病归越，筑室阳明洞，行导引术。自号"阳明子"。渐悟仙佛之非，返归儒家正统 | **【关键转折：归儒】** 从佛道归于儒学，思想路线定型 |
| 1504年（弘治十七年） | 33岁 | 返京，任兵部武选清吏司主事。与湛若水定交，相约共倡圣学 | 与湛若水结交是其思想发展的重要节点，二人共同致力于心学建设 |
| 1506年（正德元年） | 35岁 | 南京科道戴铣、薄彦徽等因弹劾刘瑾下狱。阳明上《乞宥言官去权奸以彰圣德疏》救之，触怒明武宗与刘瑾，廷杖四十，下诏狱 | **【关键转折：政治迫害】** 文人风骨之展现，政治生涯首次重大挫折。狱中读《周易》，明进退出处之道。年底贬为贵州龙场驿丞 |

## 三、贬谪贵州龙场（1506-1510）

| 时间 | 年龄 | 事件 | 思想意义/影响 |
|------|------|------|--------------|
| 1506年（正德元年）冬 | 35岁 | 廷杖四十，下诏狱。在狱中读《周易》，作《读易》诗云"瞑坐玩羲易，洗心见微奥" | 狱中读易，在绝境中寻求精神支撑，体现"致命遂志"的生命态度 |
| 1507年（正德二年）春 | 36岁 | 赴贬所。过钱塘江，刘瑾派人追杀，阳明佯投江自尽，脱险后至福建。有诗《泛海》："险夷原不滞胸中，何异浮云过太空" | 生死关头的超脱体验，加速其思想转变 |
| 1507年（正德二年）夏 | 36岁 | 抵达贵州龙场驿（今修文县）。龙场在万山丛棘中，蛊毒瘴疠，夷人�舌难语 | 从庙堂之高到江湖之远，生存困境促其反思生命意义 |
| 1508年（正德三年）春 | 37岁 | **【龙场悟道】** 日夜端居石墩（后称"玩易窝"），以生死为念。中夜忽大悟格物致知之旨，"不觉呼跃，从者皆惊"。悟得"圣人之道，吾性自足，向之求理于事物者误也" | **【关键转折：悟道】** 中国哲学史上的重大事件。心即理、心外无理的命题诞生，打破朱子学向外求理的传统，开启阳明心学新境 |
| 1508年（正德三年） | 37岁 | 在龙场建龙冈书院、宾阳堂、何陋轩、君子亭，教化当地夷人。著《五经臆说》，撰《居夷诗》 | 心学思想开始传播，书院讲学传统由此奠基 |
| 1509年（正德四年） | 38岁 | 受邀主讲贵阳文明书院，首倡"知行合一"之说。贵州提学副使席书亲往问学，叹服不已 | "知行合一"作为心学核心命题公开提出，标志其思想体系初步成型 |
| 1510年（正德五年） | 39岁 | 刘瑾伏诛。阳明升庐陵知县，结束流放生涯 | 政治转机，却无意仕途，重心转向讲学与思想传播 |

## 四、回京后的发展与事功（1510-1520）

| 时间 | 年龄 | 事件 | 思想意义/影响 |
|------|------|------|--------------|
| 1510年（正德五年） | 39岁 | 任庐陵知县，半年清理积案，政绩斐然。升南京刑部主事 | 初显治政才能，知行合一理念在实践中的验证 |
| 1511年（正德六年） | 40岁 | 入京，任吏部验封清吏司主事。与湛若水、黄绾等讲学论道 | 思想交流与传播的开始 |
| 1512年（正德七年） | 41岁 | 升吏部文选清吏司员外郎。十二月，升南京太仆寺少卿，便道归省 | 官职升迁中的讲学不辍 |
| 1513年（正德八年） | 42岁 | 在越中讲学，从学者日众。著《朱子晚年定论》，调和朱陆异同 | 思想体系的进一步完善，主动回应朱子学挑战 |
| 1514年（正德九年） | 43岁 | 升南京鸿胪寺卿。在南京与邹守益等讲学 | 南京成为其讲学的重要据点 |
| 1516年（正德十一年） | 45岁 | 升都察院左佥都御史，巡抚南赣、汀、漳等处 | 奉命平定南赣匪患，心学思想首次在军事领域全面实践 |
| 1517年（正德十二年） | 46岁 | 正月，亲率精锐屯兵上杭，佯撤实攻，连破四十余寨，斩俘七千余人。推行"十家牌法"，组织民众互保。十月，克左溪、横水，破巢八十四，斩俘六千余人。设和平县 | **【事功高峰】** 平定南赣匪患，"文臣用兵"典范。十家牌法体现其社会治理思想，军事才能震惊朝野 |
| 1518年（正德十三年） | 47岁 | 平定三浰。升都察院右副都御史。奏设和平县。在赣州濂溪书院讲学，从学者众多 | 军事成功与讲学并重，"破山中贼"之后进而"破心中之贼" |
| 1519年（正德十四年） | 48岁 | **【平定宁王之乱】** 六月十四日，宁王朱宸濠在南昌叛乱，杀巡抚孙燧。阳明正奉命往福建，闻变返江西。伪造兵部文书、施反间计迷惑宁王。七月二十六日，鄱阳湖大战，生擒宁王，前后仅43天。以万余乌合之众破十万叛军 | **【关键转折：事功顶峰】** 军事生涯顶峰，"心学战神"美誉由此而来。平叛过程中"攻心为上"的战术体现其心学智慧。事变中亲身体悟并阐发"致良知"学说 |
| 1520年（正德十五年） | 49岁 | 宁王之乱虽平，朝中权贵忌功，阳明处境艰难。武宗南巡，欲亲征宁王（已被擒），阳明抗疏谏止。六月，开讲于赣州，阐述"致良知"宗旨 | 在政治打压中完成心学理论的最后突破，"致良知"作为心学最高命题正式提出 |

## 五、晚年讲学与天泉证道（1520-1527）

| 时间 | 年龄 | 事件 | 思想意义/影响 |
|------|------|------|--------------|
| 1521年（正德十六年） | 50岁 | 明世宗即位，升南京兵部尚书。十二月，封"新建伯"，为明代以军功封爵的三位文臣之一 | 封爵象征其事功获得朝廷承认，却无意仕进，重心转向讲学 |
| 1522年（嘉靖元年） | 51岁 | 父亲王华去世，阳明守制归越，家居三年。期间讲学不辍，从学者遍天下 | 守制期间专注讲学，心学流派逐渐形成 |
| 1523年（嘉靖二年） | 52岁 | 在绍兴讲学，门人日众。编辑《传习录》上卷 | 《传习录》作为心学核心经典开始流传 |
| 1524年（嘉靖三年） | 53岁 | 建稽山书院于绍兴，四方学者云集，盛况空前。阳明以"致良知"教人，学者称"姚江学派" | 阳明心学成为与程朱理学分庭抗礼的重要流派 |
| 1525年（嘉靖四年） | 54岁 | 始辟阳明书院。门人钱德洪、王畿成为左右弟子 | 钱德洪（绪山）、王畿（龙溪）成为传播心学的主要力量 |
| 1527年（嘉靖六年）九月 | 56岁 | **【天泉证道】** 奉命出征广西前夜，于绍兴天泉桥上，与弟子钱德洪、王畿论学。阳明将毕生学问概括为"四句教"："无善无恶心之体，有善有恶意之动，知善知恶是良知，为善去恶是格物"。王畿持"四无"说，钱德洪持"四有"说，阳明判为"上根"与"中根"之教，需"相资为用，不可各执一边" | **【关键转折：教法定型】** 四句教为阳明心学的究极总结，"天泉证道"标志其思想体系完全成熟。对上根人说四无，对中根人说四有，体现其教育哲学的圆融 |

## 六、最后出征与去世（1527-1529）

| 时间 | 年龄 | 事件 | 思想意义/影响 |
|------|------|------|--------------|
| 1527年（嘉靖六年）十月 | 56岁 | 受命以南京兵部尚书兼都察院左都御史总督两广，出征广西思恩、田州平叛。途中经严滩，与弟子论学，提出"严滩问答"：有心俱是实，无心俱是幻；无心俱是实，有心俱是幻 | 严滩问答是对天泉证道的深化，体现其思想的最后定论。带病出征，践行"知行合一"于生命最后一程 |
| 1527年（嘉靖六年）十二月 | 56岁 | 抵达南宁。经考察，判断思田之乱因地方官激化矛盾所致，力主"以抚代剿" | 政治智慧的体现，否定了单纯的武力剿灭策略 |
| 1528年（嘉靖七年）正月 | 56岁 | 卢苏、王受率七万余众归降，兵不血刃平息思田之乱 | **【事功绝响】** 以德服人，展现心学在政治军事实践中的最高境界 |
| 1528年（嘉靖七年）四月 | 56岁 | 进剿断藤峡、八寨，擒斩三千五百余级。在南宁建敷文书院，兴学教化 | 军事行动与文化教化并举，"以夏变夷"的治理理念 |
| 1528年（嘉靖七年）六月 | 56岁 | 扶病撰写《祭永顺保靖土兵文》祭奠阵亡将士。旧疾（肺病）加剧，数次上疏乞骸骨，不获准 | 身体状况恶化，朝廷不允归乡 |
| 1528年（嘉靖七年）十月 | 56岁 | 疏请还乡养病，不待批复即启程 | 政治上最后的无奈抉择，也是对生命尊严的坚持 |
| 1529年（嘉靖七年十一月二十九日） | 57岁 | 1529年1月9日，病逝于江西南安府大庾县青龙港舟中。临终，弟子周积问遗言，阳明笑曰："此心光明，亦复何言？"顷之而逝 | **【生命终章】** 八字遗言浓缩其一生精神。"此心光明"——心学修养的最高境界与自我确证 |
| 1529年后 | — | 灵柩归越，"丧过江西，军民缟素哭送"。然朝廷中以桂萼为首，指其为"伪学"，夺爵禁学。至隆庆元年（1567）始平反，追赠新建侯，谥"文成"。万历十二年（1584）从祀孔庙 | 身后毁誉参半，却无法阻挡其思想的传播。心学流播至日本、朝鲜半岛，影响东亚思想史五百年 |

## 八、关键节点标注

### ★ 生平三大转折

1. **1489年（18岁）——立志成圣**：识娄谅，信"圣人必可学而致之"，圣人之志确立
2. **1508年（37岁）——龙场悟道**："圣人之道，吾性自足"，心即理、心外无理命题诞生
3. **1527年（56岁）——天泉证道**：四句教概括毕生学问，心学体系成熟定型

### ★ 事功三大节点

1. **1517-1518年（46-47岁）——平定南赣**：平定赣南、汀、漳匪患，建立十家牌法，改土归流
2. **1519年（48岁）——平定宁王之乱**：43天平叛，生擒朱宸濠，"心学战神"美誉
3. **1528年（56岁）——思田之役**：兵不血刃，以抚代剿，心学治道的典范

## 九、思想发展谱系

```
早年探索（1472-1506）
    └─ 格竹失败（1492）→ 质疑向外求理
    └─ 出入佛老（1500-1502）→ 最终归于儒学

龙场悟道（1506-1510）
    └─ 生死体验 → 超脱得失荣辱
    └─ 中夜大悟（1508）→ 心即理、心外无理
    └─ 龙冈讲学 → 思想初步传播
    └─ 知行合一（1509）→ 核心命题提出

思想成熟（1510-1520）
    └─ 事功实践 → 心学的政治军事验证
    └─ 致良知（1519-1520）→ 最高命题确立
    └─ 讲学传道 → 姚江学派形成

晚年定型（1520-1529）
    └─ 天泉证道（1527）→ 四句教总结
    └─ 严滩问答 → 思想最后定论
    └─ 此心光明（1529）→ 生命境界的自我确证
```
