# 孔子的对话与即兴思考

## 一、"同问不同答"：因材施教的对话艺术

### 1. 问仁的多重回答

**【一手】《论语》原文记载的"问仁"对话**

《论语》中共有十三则弟子向孔子问仁的对话，主要包括：

| 问者 | 出处 | 孔子回答 | 特点分析 |
|------|------|----------|----------|
| 颜渊 | 颜渊12.1 | "克己复礼为仁。一日克己复礼，天下归仁焉。" | 最深层次的回答，给出具体条目：非礼勿视、非礼勿听、非礼勿言、非礼勿动 |
| 仲弓（冉雍） | 颜渊12.2 | "出门如见大宾，使民如承大祭。己所不欲，勿施于人。在邦无怨，在家无怨。" | 从政角度，强调敬业与推己及人 |
| 司马牛 | 颜渊12.3 | "仁者其言也讱。" | 针对其多言躁动的性格，强调言语谨慎 |
| 樊迟（第一次） | 颜渊12.22 | "爱人。" | 简洁，强调仁的普遍性 |
| 樊迟（第二次） | 子路13.19 | "居处恭，执事敬，与人忠。" | 处世态度角度 |
| 樊迟（第三次） | 颜渊12.21 | "仁者先难而后获，可谓仁矣。" | 从功利角度引导 |

**可信度：高**（一手来源，论语原文）

**【二手】后人对"问仁"不同回答的解读**

来源：知猫论文《从〈论语〉中弟子问仁探究颜渊地位》
- 孔子对颜渊问仁的回答最为系统完整
- 不同回答体现孔子对弟子性格的了解和针对性教学
- 因人、因时、因地、因事而异其说

可信度：中（学术论文，但需要结合原文理解）

### 2. 问孝的四重境界

**【一手】《论语·为政》四种问孝**

| 问者 | 原文 | 出处 |
|------|------|------|
| 孟懿子 | "无违。" | 为政2.5 |
| 孟武伯 | "父母唯其疾之忧。" | 为政2.6 |
| 子游 | "今之孝者，是谓能养。至于犬马，皆能有养；不敬，何以别乎？" | 为政2.7 |
| 子夏 | "色难。" | 为政2.8 |

**【二手】朱熹《论语集注》解释**

程颐解释："告懿子，告众人者也。告武伯者，以其人多可忧之事。子游能养而或失于敬，子夏能直义而或少温润之色。各因其材之高下，与其所失而告之，故不同也。"

**可信度：高**（经典注疏）

### 3. "闻斯行诸"：同一问题当场给出相反回答

**【一手】《论语·先进》11.22原文**

```
子路问："闻斯行诸？"子曰："有父兄在，如之何其闻斯行之？"
冉有问："闻斯行诸？"子曰："闻斯行之。"
公西华曰："由也问闻斯行诸，子曰'有父兄在'；求也问闻斯行诸，子曰'闻斯行之'。赤也惑，敢问。"
子曰："求也退，故进之；由也兼人，故退之。"
```

**核心机制**：孔子明确解释了他的判断逻辑——"求也退（冉有谦退），故进之（鼓励他行动）；由也兼人（子路勇猛过人），故退之（让他谨慎）。"

**可信度：高**（一手原文，且孔子本人解释了原因）

**对话特点分析**：
1. 当场给出相反答案，需要极强的观察力和判断力
2. 孔子主动解释差异原因，展现教育自觉
3. 公西华的困惑代表旁观者视角

## 二、对话对象分类：弟子、君主、隐士

### 1. 与弟子的对话

**子路：最频繁的对话对象**

《论语》中孔子与子路的对话出现**30次**，孔子对子路的评价**26次**，其中批评**20次**，表扬**8次**。

来源：知猫论文《〈论语〉中孔子批评观照下的子路形象分析》
可信度：中（统计数据可能存在不同统计标准）

**【一手】典型对话场景**

| 场景 | 出处 | 孔子回应特点 |
|------|------|--------------|
| 子路问政 | 子路13.1 | "先之，劳之。"——简短，点到即止；追问后答"无倦" |
| 子路问君子 |宪问14.42 | 层层追问，孔子逐一回答，形成"修己以敬→修己以安人→修己以安百姓"的递进 |
| 子路问事君 | 先进11.24 | "勿欺也，而犯之。"——直接、原则性强 |
| 子路使子羔为费宰 | 先进11.25 | 孔子批评"贼夫人之子"，子路反驳，孔子说"是故恶夫佞者" |

**【矛盾记录】**

子路评价存在矛盾：
- 赞扬："道不行，乘桴浮于海。从我者，其由与？"（公冶长5.7）
- 批评："野哉由也！"（子路13.3）
- 评价："由也升堂矣，未入于室也。"（先进11.15）

**处理**：不调和。这些矛盾反映了孔子对子路的多角度观察和不同情境下的不同评价。

**颜渊：最得意的弟子**

**【一手】孔子对颜渊的评价**

| 出处 | 原文 | 场景 |
|------|------|------|
| 雍也6.3 | "贤哉回也！一箪食，一瓢饮，在陋巷，人不堪其忧，回也不改其乐。贤哉回也！" | 赞美安贫乐道 |
| 为政2.9 | "吾与回言终日，不违，如愚。退而省其私，亦足以发，回也不愚。" | 从误解到认可 |
| 雍也6.7 | "回也，其心三月不违仁，其馀则日月至焉而已矣。" | 仁德修养评价 |
| 先进11.19 | "回也其庶乎，屡空。"——注：对此句有不同解读 | 贫穷与道德 |

**【争议】"屡空"的解读**

- 一说：颜回常常贫穷空匮
- 二说：颜回每每虚心（空=虚中）

**可信度：高**（原文记载）但解读存在分歧

**宰我：最受批评的弟子**

**【一手】宰我昼寝事件**

《论语·公冶长》5.10：
```
宰予昼寝。子曰："朽木不可雕也，粪土之墙不可杇也。于予与何诛？"
子曰："始吾于人也，听其言而信其行；今吾于人也，听其言而观其行。于予与改是。"
```

**可信度：高**（一手原文）

**【一手】宰我问三年之丧**

《论语·阳货》17.21：
```
宰我问："三年之丧，期已久矣。君子三年不为礼，礼必坏；三年不为乐，乐必崩。旧谷既没，新谷既升，钻燧改火，期可已矣。"
子曰："食夫稻，衣夫锦，于女安乎？"
曰："安。"
"女安则为之！夫君子之居丧，食旨不甘，闻乐不乐，居处不安，故不为也。今女安，则为之！"
宰我出。
子曰："予之不仁也！子生三年，然后免于父母之怀。夫三年之丧，天下之通丧也。予也有三年之爱于其父母乎？"
```

**对话技巧分析**：
1. 孔子问"安乎"——从内心感受入手
2. 宰我答"安"——孔子不强行反驳
3. 宰我离开后孔子才评价——不在当面攻击

**可信度：高**（一手原文）

### 2. 与君主的对话

**【一手】与鲁定公的对话**

《论语·八佾》3.19：
```
定公问："君使臣，臣事君，如之何？"
孔子对曰："君使臣以礼，臣事君以忠。"
```

**对话特点**：
- 回答简洁（8字）
- 使用对偶结构，形式工整
- 不立不破，强调双向责任

**可信度：高**

**【一手】与齐景公的对话**

《论语·颜渊》12.11：
```
齐景公问政于孔子。
孔子对曰："君君，臣臣，父父，子子。"
公曰："善哉！信如君不君，臣不臣，父不父，子不子，虽有粟，吾得而食诸？"
```

**对话技巧**：
- 六字四叠，极简形式
- 齐景公自己得出结论
- 历史背景：当时陈氏专政、景公宠爱少子逐长子（见《论语述何》）

**可信度：高**（原文）；历史背景解释可信度：中（后人推断）

**【一手】与鲁哀公的对话**

《论语·为政》2.19：
```
哀公问曰："何为则民服？"
孔子对曰："举直错诸枉，则民服；举枉错诸直，则民不服。"
```

**对话特点**：
- 使用对比手法
- 正反两方面
- 简明原则

**可信度：高**

### 3. 与隐士的对话

**【一手】楚狂接舆**

《论语·微子》18.5：
```
楚狂接舆歌而过孔子曰："凤兮凤兮，何德之衰！往者不可谏，来者犹可追。已而已而，今之从政者殆而！"
孔子下，欲与之言。趋而辟之，不得与之言。
```

**关键**：孔子"欲与之言"但对方拒绝——对话未能发生

**可信度：高**

**【一手】长沮、桀溺**

《论语·微子》18.6（节选）：
```
长沮、桀溺耦而耕。孔子过之，使子路问津焉。
长沮曰："夫执舆者为谁？"子路曰："为孔丘。"曰："是鲁孔丘与？"曰："是也。"曰："是知津矣。"
...
桀溺曰："...滔滔者天下皆是也，而谁以易之？且而与其从辟人之士也，岂若从辟世之士哉？"
...
子路行以告。夫子怃然曰："鸟兽不可与同群，吾非斯人之徒与而谁与？天下有道，丘不与易也。"
```

**对话结构**：
1. 子路问津→隐士嘲讽
2. 隐士提出"避世"主张
3. 孔子回应"入世"立场

**可信度：高**

**【一手】荷蓧丈人**

《论语·微子》18.7（节选）：
```
子路从而后，遇丈人，以杖荷蓧。
子路问曰："子见夫子乎？"
丈人曰："四体不勤，五谷不分，孰为夫子？"植其杖而芸。
...
子路以告。孔子曰："隐者也。"使子路反见之。至则行矣。
```

**关键**：丈人批评孔子"四体不勤，五谷不分"——孔子派人回去见他时，他已离开

**可信度：高**

## 三、拒绝回答的问题

### 1. "子不语怪力乱神"

**【一手】原文**

《论语·述而》7.21：
```
子不语怪、力、乱、神。
```

**【二手】经典注疏解释**

王肃（三国）注：
- 怪：怪异之事
- 力：若奡荡舟、乌获举千钧之属（超常力量）
- 乱：臣弑君、子弑父
- 神：鬼神之事

朱熹引谢氏："圣人语常而不语怪，语德而不语力，语治而不语乱，语人而不语神。"

**可信度：高**（原文及经典注疏）

### 2. "未知生，焉知死"

**【一手】原文**

《论语·先进》11.12：
```
季路问事鬼神。
子曰："未能事人，焉能事鬼？"
曰："敢问死。"
曰："未知生，焉知死？"
```

**对话策略分析**：
- 不直接否定问题
- 用反问引导
- 把问题引回到现实人生

**可信度：高**

### 3. 子贡问死

**【一手】类似场景**

《论语》中孔子对鬼神、死亡等问题多采取回避态度。

## 四、改变立场/判断的瞬间

### 1. 对宰我的判断改变

**【一手】**

《公冶长》5.10：
```
"始吾于人也，听其言而信其行；今吾于人也，听其言而观其行。于予与改是。"
```

**核心**：因为宰我，孔子改变了对人的判断方式——从"听其言信其行"到"听其言观其行"

**可信度：高**

### 2. 对颜回的判断转变

**【一手】**

《为政》2.9：
```
"吾与回言终日，不违，如愚。退而省其私，亦足以发，回也不愚。"
```

**演变**：初以为愚→观察后发现"足以发"→不愚

**可信度：高**

### 3. "何必读书"之争

**【一手】**

《先进》11.25：
```
子路使子羔为费宰。
子曰："贼夫人之子。"
子路曰："有民人焉，有社稷焉，何必读书，然后为学？"
子曰："是故恶夫佞者。"
```

**孔子立场**：坚持读书为学的重要性，但以"恶夫佞者"回应子路的辩驳，情绪明显。

**可信度：高**

## 五、争议性对话与不同解读

### 1. "民可使由之，不可使知之"

**【一手】原文**

《论语·泰伯》8.9：
```
子曰："民可使由之，不可使知之。"
```

**【争议】断句与理解**

传统断句："民可使由之，不可使知之。"——民众可以让他们照着做，不可以让他们知道为什么。
- 理解1：愚民政策（批评者观点）
- 理解2：民众知识有限，难以理解高深道理（辩护者观点）

新式断句：
- "民可，使由之；不可，使知之。"——民众可以做到的，让他们照做；做不到的，教导他们。
- "民可使，由之；不可使，知之。"

**可信度：原文高，解读争议大**

### 2. "唯女子与小人为难养也"

**【一手】原文**

《论语·阳货》17.25：
```
子曰："唯女子与小人为难养也，近之则不孙，远之则怨。"
```

**【争议】**

- 传统解读：对女性和地位低下者的贬低
- 辩护解读："女子"指特定身份的女性（如姬妾）、"小人"指特定人群
- 历史主义解读：春秋时代的观念，不宜以现代标准苛责

**可信度：原文高，解读争议大**

### 3. "父亲偷羊，儿子作证"之争（亲亲相隐）

**【一手】原文**

《论语·子路》13.18：
```
叶公语孔子曰："吾党有直躬者，其父攘羊，而子证之。"
孔子曰："吾党之直者异于是。父为子隐，子为父隐，直在其中矣。"
```

**【争议】**

- 赞同方：认为此乃亲情优先、家庭伦理至上
- 批评方：认为此主张损害法治、鼓励包庇
- 中间立场：认为"隐"不等于包庇犯罪，而是家庭伦理优先于司法伦理

**可信度：原文高，评价争议大**

## 六、即兴思考的特征总结

### 1. 形式特征

| 特征 | 体现 | 来源 |
|------|------|------|
| 简短 | 多数回答不超过20字 | 《论语》通篇 |
| 对偶 | "君君臣臣父父子子"、"君使臣以礼，臣事君以忠" | 多处 |
| 反问 | "未知生，焉知死"、"女安乎？" | 先进、阳货 |
| 隐喻 | 用日常事物比喻（凤鸟、稻锦） | 多处 |

### 2. 内容策略

| 策略 | 案例 | 效果 |
|------|------|------|
| 因人而答 | 闻斯行诸 | 同一问题相反答案 |
| 引而不发 | 问死、问鬼神 | 推回现实问题 |
| 情境切入 | 问仁、问孝 | 针对具体短板 |
| 先肯定再纠正 | 宰我问丧 | "女安则为之"→背后批评 |

### 3. 思维模式

**根据二手资料（《国学知识·从孔子〈论语〉对话风采看文艺批评》）总结**：

孔颖达在《论语集解义疏序》中分析：
"此书之体，适会多途，皆夫子平生应机作教，事无常凖，或与时君抗厉，或共弟子抑扬，或自显示物，或混迹齐凡，问同荅异，言近意深。"

**核心**：
- "应机作教"——随机性
- "事无常凖"——无固定规则
- "问同荅异"——同问不同答
- "言近意深"——浅近话语蕴含深意

## 八、矛盾记录与待考证问题

### 已发现矛盾

1. **对子路的评价矛盾**
   - 赞扬与批评并存
   - 记录：不调和，视为孔子对不同情境的不同反应

2. **"屡空"解读分歧**
   - 一说贫穷，一说虚心
   - 记录：保留两种可能

3. **孔子的情绪表达**
   - 有时平和（问仁、问孝）
   - 有时激烈（"朽木不可雕"、"野哉由也"）
   - 解读：不同情境、不同对象的自然反应

### 待考证

1. 部分对话的具体历史背景（如与齐景公对话前的陈氏专政背景）
2. 弟子性格的历史记载来源
3. 部分注疏的时代局限性
