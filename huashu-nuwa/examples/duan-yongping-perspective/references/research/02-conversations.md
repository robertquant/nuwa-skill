### 1. 2025年雪球《方略》访谈（方三文 × 段永平）
**来源**: 雪球平台、澎湃新闻、36氪等  
**时长**: 93分钟（完整版约2小时）  
**性质**: 一手（段永平本人主讲）  
**URL**: 
- https://xueqiu.com/8434340331/361060965
- https://www.thepaper.cn/newsDetail_forward_31947522
- https://www.36kr.com/p/3549591336819078

**访谈背景**: 这是段永平"退休"二十多年后的罕见公开对话，于2025年10月16日在美国加州进行。

**关键对话片段（被追问时的回应方式）**:

1. **关于成长环境与成就的关系**
   - 方三文：小时候的成长环境，和现在的性格、成就有关系吗？
   - 段永平：**"这个就不知道了。"** 然后简单讲述经历后就不说话了。
   - 方三文继续追问：家庭环境和父母教育方式是什么样的？
   - 段永平：**"这些东西其实没有太大意思。我在哪条河里摸过鱼，对他能有什么帮助？我很难讲得清楚。"**
   - 分析：**直接承认不知道/没意义，拒绝过度解读。这体现了他的思维边界——不编造因果逻辑。**

2. **关于"看懂"的定义**
   - 问：如何快速判断一家公司是否值得投资？
   - 段永平：**"基本上，我不会快速做出这样的判断。"**
   - 后续解释：过去十多年，他关注的公司就那几个，源自多年对企业、生意、产品、所有东西的理解积攒。
   - f="**"我没见过谁真的能够快，包括沃伦巴菲特也好，芒格也好，其实投资做得好的人都很慢。大家并不在乎失去一些机会，但是最重要的是你不要去踩雷。"**

3. **关于茅台暴跌时的反应**
   - 访谈中最震撼的不是他怎么赚到钱的，而是他在茅台暴跌50%时说的那句话。
   - 这是段永平投资哲学的核心体现：坚守能力圈内的信念。

4. **关于"懂与不懂"的边界**
   - 段永平用一句话界定边界：**"一字不懂不等于不挣钱，但真懂了才敢重仓。"**
   - 他说自己20年来认真投资过的公司不超过10个，因为只有这些是他真正"看懂"的。

5. **关于巴菲特的启示**
   - 问：从巴菲特那学到的最重要的东西是什么？
   - 段永平：**"买股票就是买公司"** —— 这一句话就够了。
   - 巴菲特后来回复他："这也正是我当年跟本杰明·格雷厄姆那儿学到的最重要的一句话。"

**访谈中他拒绝回答的问题类型**:
1. 具体的价格预测和估值目标
2. 对其他投资人的评价
3. 短期市场走势判断
4. 个人生活细节的因果归因

## 二、被追问时的回答方式分析

### 1. 直面但拒绝过度解读
**特点**: 承认不知道，但会给出基本信息，然后停止编造因果。

**案例**:
- 问：成长环境与成就的关系？
- 答："这个就不知道了。" + 简述经历 + 沉默。
- 他不编造"小时候摸鱼经历培养了商业思维"这种因果链。

### 2. 用底层逻辑统一回答
**特点**: 不同问题用同样的核心原则回答，不逐一分析表面现象。

**案例**（浙大演讲）:
- 学生问AI趋势、创业选择、学习方法等问题
- 他几乎都回答类似的逻辑：**"想5年、10年、20年以后的决策对不对。"**

### 3. 明确边界，坦承看不懂
**特点**: 不懂的领域直接说不懂，不装懂也不敷衍。

**案例**（雪球访谈）:
- 访谈中提到10次「犯错」、12次「看懂」、29次「很难」
- **"买股票就是买公司这句话，能有1%的人真懂这句话就了不起，做到就更难。"**

### 4. 拒绝预测和估值目标
**特点**: 不谈具体价格、不从市盈率角度给建议。

**案例**:
- 问估值问题，他几乎从不谈具体的市盈率目标、目标价格或精确的DCF输入假设
- 更常说的是："看懂未来现金流"——本质思维而非精确计算

## 三、即兴类比和隐喻

### 1. "刮瓶盖"类比
**场景**: 解释为什么离开小霸王
**原文**: **"你刮瓶盖刮出谢了，还会继续刮吗？"**
**含义**: 发现承诺不可信，就没必要继续纠缠。这是他"不合适立刻离开"决策逻辑的生动表达。

### 2. "收费站"类比
**场景**: 向巴菲特解释苹果的商业模式
**原文**: 苹果的生意模式就像一个收费站
**效果**: 巴菲特一句话就明白了苹果的价值所在

### 3. "打孔机"类比
**场景**: 解释投资集中度
**原文**: **"一辈子20个投资额度，我目前还没打完10个孔。"**
**含义**: 重大投资决策要极度慎重，重仓标的屈指可数

### 4. "种田vs狩猎"类比
**场景**: 区分投资与投机
**原文**: **投资像"种田"，靠企业成长收获；投机像"狩猎"，零和博弈、互为猎物。**

### 5. "安全感"类比
**场景**: 解释父母教育的影响
**原文**: "我小时候父母不怎么管我......我觉得我还是比较有安全感的。"
**延伸**: 这种安全感可能是他敢于做自己、不从众的底色

## 四、拒绝回答的问题（揭示边界）

### 1. 具体的价格预测
- 不预测目标价、市盈率倍数
- 认为这些讨论是"瞎扯"

### 2. 短期市场走势
- 不预测几个月或一年内的市场
- 关注的是5年、10年、20年的维度

### 3. 对其他投资人的评价
- 基本不公开点评其他投资者
- 最多说"大家各有各的方法"

### 4. 个人生活的因果归因
- 拒绝将个人成就归因于具体童年经历
- 认为这类"成功学"问题"没有太大意思"

### 5. 看不懂的公司
- 明确说"看不懂10年后会怎样"的公司，不碰
- 典型案例：泡泡玛特（初期）、人形机器人

## 八、矛盾记录

### 1. 关于"长期持有"
- **矛盾点**: 外界认为价值投资=长期持有不卖
- **段永平澄清**: **"投资是要卖的，我觉得很多人对所谓的价值投资有个很大的误解就是，你要长期持有，就不可以卖了。"**
- **他的立场**: "长期持有是一个意向，我买的那个刹那是打算长期持有的，但是你永远要算你的机会成本。"

### 2. 关于"满仓主义" vs "空仓快感"
- **矛盾点**: 曾说自己是"满仓主义者"，又说享受"空仓的快感"
- **分析**: 这不是矛盾，而是基于机会成本的动态决策——没有好标的时空仓也舒服，有好标的时满仓也舒服

### 3. 关于泡泡玛特（立场改变）
- **初始立场**: "看不懂""不会投资"
- **后来立场**: "Right business, Right people" + 实际建仓
- **他自己的解释**: 通过新的学习和调研改变了看法
- **这是真正的立场改变，不是前后矛盾**

## 十、调研总结

### 段永平在被追问时的核心特点

1. **敢于说"不知道"**: 不编造因果，不强行归因
2. **用底层逻辑统一回答**: 不同问题回归同一核心原则
3. **明确边界**: 看不懂就是看不懂，不装懂
4. **拒绝预测**: 不给价格目标、市场点位
5. **愿意改变立场**: 泡泡玛特案例展示了他的开放性

### 他的即兴类比风格

1. **生活化**: 刮瓶盖、打孔机、种田狩猎
2. **简洁有力**: 一句话就能让巴菲特"明白"
3. **本质导向**: 类比指向核心问题，而非表面相似

### 他拒绝回答的问题类型

1. 具体价格预测和估值目标
2. 短期市场走势判断
3. 对其他投资人的评价

## 十一、《方略》访谈完整实录精选（2025年，澎湃新闻）

**来源**: 澎湃新闻（全文约2万字）  
**时长**: 近2小时  
**性质**: 一手（段永平本人主讲）  
**URL**: https://www.thepaper.cn/newsDetail_forward_31947522

这是段永平"退休"二十多年后的罕见公开对话，与雪球创始人方三文进行。以下是关键对话片段的深度分析：

### 被追问时的回答方式

#### 案例1：成长环境与成就的关系
```
方三文：小时候你是在什么样的环境下长大的？这个成长的环境跟你现在的性格、成就有没有关系？

段永平：这个就不知道了。我是在南昌出生的，六岁左右跟父母下放到江西安福县……（简述经历）确实吃过苦……

方三文追问：你能不能描述一下你的家庭环境，以及父母对你施加了怎么样的教育？

段永平：这些东西其实没有太大意思。我在哪条河里摸过鱼，对他能有什么帮助？我很难讲清楚。
```

**分析**：
- **第一次回答**：直接说"这个就不知道了"，然后给出基本信息后停止
- **被追问后**：明确指出这类问题"没有太大意思"，拒绝编造因果
- **即兴类比**："我在哪条河里摸过鱼，对他能有什么帮助？"——用童年摸鱼的经历类比成功学归因的荒谬
- **边界清晰**：不把童年经历过度解读为成就的原因

#### 案例2：关于"懂与不懂"的边界
```
方三文：怎么样确定自己搞懂了一个公司？

段永平：我本身做游戏出身，我跟他们游戏团队聊聊，觉得他们是一帮很热爱游戏的人……这个商业模式，我觉得它很有道理，加上它的现金当时比市值高。我作为风投一样的，买进去很简单。

方三文追问：对网易的判断里面，听起来包括了两个部分，一是你对游戏这个商业本身有个判断？

段永平：当然，我做游戏出身的，我可懂游戏了。

方三文追问：第二，对它的定价的判断，比如说它的股价很低，可能低于它持有的净现金。这个其实也是巴菲特说的安全边际？

段永平：巴菲特的安全边际不是指这个，安全边际指的是你对公司有多懂，这是我的理解。不是说有多便宜，便宜的东西可以更便宜。

方三文追问：懂和不懂，按你这么说，其实它处于一个灰色的区域，不是那么清晰的一个标准，是这样吗？

段永平：我也不知道，但是到处问人家的人肯定是不懂的。那么不问的人是不是就懂了，你可以说它属于灰色地带。至少我赚了100多倍，你说我懂还是不懂，我卖肯定有我卖的原因，有卖的道理，而且我其实也没少赚。
```

**分析**：
- **层层追问下的态度变化**：从"我懂游戏"的确定，到"安全边际是对公司有多懂"的定义纠正，再到"我也不知道"的坦承边界
- **不装懂**：承认"懂"是灰色地带，但明确说"到处问人家的人肯定是不懂的"
- **用结果说话**："我赚了100多倍，你说我懂还是不懂"——用结果而非定义来回应

#### 案例3：关于巴菲特的启示
```
段永平：后来直到我看了巴菲特的东西，其实啥也没看，就看"买股票就是买公司"，我就看到那一句话，突然就明白了，这句话就够了。

方三文：如果用一句话来概括你的投资理念的话，是不是买股票就是买公司？

段永平：是的。但是后面那句话是，你要看懂公司，这个很难。为什么说投资很简单，但是不容易？简单就是你一定要看公司，你一定要看懂生意，你也要看懂未来的现金流；难是难在你很难做到这一点，大部分公司都不容易看懂。
```

**分析**：
- **即兴提炼**："投资很简单，但是不容易"——这是一个经典的段永平式表达
- **用对比结构回答**：先确认核心观点，再用"但是"补充关键的困难

### 即兴类比和隐喻（补充案例）

#### 1. "刮瓶盖"类比（完整版）
```
段永平：你跟我讲什么都是不可信的。你有一次两次，难道还要有第三次吗？就像你刮那个瓶盖，刮出"谢"了，你还继续刮完它吗？两个字你刮出一个字，肯定就不往下刮了，我就是刮到那个字后觉得肯定不能再待下去了。
```
**含义**：契约精神被破坏一次，就不应该再等待第二次。用生活中的刮瓶盖场景，把复杂的决策逻辑讲得极其通俗。

#### 2. "收费站"类比（对巴菲特）
```
段永平：我对他们讲，苹果的生意模式就像一个收费站。
```
**效果**：巴菲特一句话就明白了苹果的价值所在。这是即兴表达的极致——用一个词让对方完全理解。

#### 3. "开车的方向vs看后视镜"
```
段永平：乔布斯跟蒂姆·库克讲，你当CEO就是你做决策，你不要想我乔布斯会做什么，这就对了。我也是这样跟大家讲……松下的社长跟我讲"当我在做一个决策的时候，我就想着松下老人站在我背后会怎么想"，我心想完蛋了，因为松下老人早就不在了。

方三文：它可能把自己过往的体制、成就，看得太重了？

段永平：这个我不知道。你的眼睛是要看到用户的，他们的眼睛是看倒后镜的。等时间足够长，一定是会出毛病的。
```
**含义**：用"开车看后视镜"类比企业决策只关注过去、不看用户需求的问题。生动且精准。

#### 4. "北斗星"类比
```
段永平：文化好，最主要的就是最终它会走回正道，有一个北斗星在指引着它，它到底该干什么。不是为了生意，你仅仅讨论生意是很容易犯错误的。
```

#### 5. "高尔夫比喻"
```
方三文：要搞懂不变的东西，或者搞懂创新的东西，其实都挺难？

段永平：当然，搞懂什么都难，但是再难也没有高尔夫难。
```
**风格**：用幽默的夸张结束话题，不继续纠缠"难不难"的问题。

### 拒绝回答的问题类型（完整分析）

#### 1. 具体的价格预测和估值目标
```
方三文：你如何快速判断一家公司是否值得投资？

段永平：基本上，我不会快速做出这样的判断。
```
**分析**：直接否定问题的前提（"快速判断"），将问题导向自己的逻辑框架。

#### 2. 个人生活的因果归因
贯穿访谈的核心态度：
- 不编造"童年经历培养商业思维"这类因果链
- 多次强调"我不知道""没有太大意思"
- 拒绝把个人成功归因于具体细节

#### 3. 对其他投资人的评价
```
方三文：你对彼得·林奇怎么看？

段永平：（只提数据，不做评价）彼得·林奇，他一辈子做过2000多个股票，他一定很忙，所以头发白得也很早，早早的就退休了。
```
**分析**：用客观数据回应，避免主观评价。

#### 4. 预测问题
```
方三文：茅台将来一定会回来吗？

段永平：我觉得茅台将来一定会回来，但是什么时候我不知道，可能五年十年，没有人知道。
```
**分析**：给结果预测（"会回来"），但拒绝给时间预测（"什么时候我不知道"）。

### 立场改变的瞬间（补充案例）

#### 关于茅台的持有心态
```
段永平：其实2600、2700块的时候我可想卖了，因为觉得相对来讲那个时点是贵的，我当时最犹豫的就是，我们的仓位太大了，我卖了以后我买什么？我就开始折磨自己，然后放眼看去我什么都买不下手。

事后诸葛亮，你也没想到它会一下掉那么多，从2600掉到1200块，掉了50%，其实是没多少。一个股票，你要是扛不住掉50%，你就不应该买。我觉得是OK的，所以我们后来其实还买了不少。
```

**分析**：
- 坦承自己有过动摇（"我可想卖了"）
- 解释为什么没卖的真正原因（机会成本，不是坚定持有）
- 承认事后看是"事后诸葛亮"
- 用"扛不住掉50%就不应该买"定义持有标准

#### 关于"长期持有"的澄清
```
段永平：投资是要卖的，我觉得很多人对所谓的价值投资有个很大的误解就是，你要长期持有，就不可以卖了。长期持有是一个意向，我买的那个刹那我是打算长期持有的，但是你永远要算你的机会成本。
```
**这是重要的立场澄清**：打破了外界对"价值投资=永远不卖"的误解。

### 对争议话题的直接回应

#### 关于马斯克和特斯拉
```
段永平：确实觉得埃隆·马斯克这个人厉害，他有很多想法，也确实是很先进的。但是投资我觉得比较难，从个人的角度来讲，我不是很喜欢这个人的品行。你投资实际上是在跟他做朋友，我不想跟他做朋友，哪怕给我钱我也不干。所以我没有那么喜欢这件事情，但是我认识到他确实是厉害。

方三文：你觉得他的生意怎么样？

段永平：电动车这个生意不会太好，会很累，它差异化很小。但是特斯拉的电动车，它是做出了差异化的，大部分电动车的生意我觉得会很艰苦，特斯拉总体来讲确实做得好。

我很早就在博客上写过特斯拉，我最早是很喜欢特斯拉的，你看我的车牌都是"我们爱特斯拉"。后来就看马斯克那些事，我就觉得有点不太喜欢这个人，加上我买了他们车以后，看服务各方面，好多我没那么满意的东西，又把股票也卖了。当然这是个错误的决策，我可能应该留着，但是我觉得很难，确实很难。
```

**风格分析**：
- 先肯定对方的优点（"明确厉害"）
- 再表达自己的立场（"不喜欢这个人的品行"）
- 解释投资逻辑（"投资实际上是跟他做朋友"）
- 坦承自己的错误决策（"这是个错误的决策"）

#### 关于通用电气的错误投资
```
段永平：犯错误有时候也是会挣钱的，我觉得投通用电气是一个错误，回到今天我不会买。

方三文：即使挣了钱你也不能否认它是个错误？

段永平：它当然是个错误，因为它不符合我后来决定的，要符合企业文化和商业模式这两个过滤器。
```
**核心观点**：挣钱和错误可以同时存在——关键是不符合投资框架。

### 关键对话模式总结

1. **追问时的态度变化**：从确定到承认灰度，但始终不装懂

3. **即兴类比的特征**：生活化、一句话说透本质
4. **对争议的回应**：先肯定合理部分，再表达立场，最后坦诚局限
5. **纠正误解的方式**：用重新定义打破刻板印象（"长期持有≠不卖"）
