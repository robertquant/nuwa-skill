# 段永平人生时间线

> 调研时间：2026年4月
> 信息来源：访谈、演讲、自述（一手）、媒体报道（二手）

## 一、出生与求学（1961-1988）

| 时间 | 事件 | 信息来源 |
|------|------|----------|
| **1961年3月10日** | 出生于江西南昌。父母均为大学教师（父亲在南昌工程学院任教）。童年生活无忧。 | 二手（今日头条、豆粒网） |
| **约1967年** | 约6岁时随父母下放到江西安福县。 | 二手（百科资料） |
| **1977年10月** | 国家恢复高考。首次参加高考，考80多分（满分500分），名落孙山。 | 二手（百科、知乎年谱） |
| **1978年7月** | 复读一年后，以400多分（五门课平均80分以上）考入浙江大学无线电系。成为恢复高考后首届大学生。 | 二手（百科、媒体报道） |
| **1978年10月** | 进入浙江大学无线电系就读，时年17岁。 | 二手（历史资料整理） |
| **约1980年** | 大学期间曾休学一年（具体原因不详）。 | 二手（豆瓣等资料提及） |
| **1982年/1983年** | 浙江大学毕业（因休学延迟一年）。被分配到北京电子管厂（国企），任技术员，月薪46元。这是当时人人羡慕的"铁饭碗"。 | 二手（百科、访谈记录） |
| **约1983-1985年** | 北京电子管厂工作期间，工资逐步上涨至80多元/月。干了约三年，感到厌倦，决定考研。期间曾有第一次商业尝试——贩卖章光101生发剂，虽未赚大钱，但激发从商兴趣。 | 二手（今日头条时间线） |
| **1986年** | 考入中国人民大学经济系，攻读计量经济学研究生。 | 二手（百科、多处报道一致） |
| **1988年7月** | 从人大研究生毕业。因未提交学位论文，最高学历为学士。有人建议请人代写论文，他拒绝了。毕业时有机会留京，但选择南下广东。 | 二手（多处报道一致） |

## 二、小霸王时期（1988-1995）

| 时间 | 事件 | 信息来源 |
|------|------|----------|
| **1988年下半年** | 南下广东，被中山市怡华集团聘任。具体角色有不同说法。 | 二手（百科资料） |
| **1989年3月** | 正式接手中山市怡华集团下属日华电子厂（小厂），该厂年亏损200万元，出任厂长。时年28岁。是段永平商业传奇的起点。 | 二手（百科、媒体报道一致） |
| **1989-1990年** | 敏锐洞察到中国家庭电视游戏机市场的巨大潜力。1990年中国内地卖出300万台游戏机，预测饱和容量达4000万台。决定转型生产家用电视游戏机。 | 二手（知乎、B站分析） |
| **1991年** | 将工厂更名为"中山市小霸王电子工业公司"，正式创立"小霸王"品牌。 | 二手（百科） |
| **1991年** | 投入40万元拍下央视黄金时段广告，开创国产电子品牌央视投放先河。"拥有一台小霸王，打出一个万元户"广告语家喻户晓。 | 二手（网易、搜狐） |
| **1993年** | 推出"小霸王学习机"，将游戏与学习结合，精准契合"望子成龙"社会心理。邀请成龙代言，"同是天下父母心，望子成龙小霸王"广告深入人心。 | 二手（媒体报道一致） |
| **1994年** | 小霸王年产值突破10亿元，市场份额近80%，成为中国最知名的电脑品牌（超过联想和IBM）。段永平被称为"打工皇帝"，但仍只是高级打工仔。 | 二手（多个来源一致） |
| **1994年** | 向怡华集团提出股份制改造方案，承诺初始阶段股权"三七开"（段永平团队30%）。但被集团大股东否决。 | 二手（网易报道） |
| **1995年** | 多次提出股份制改造均被否决，承诺不能兑现，决定离开。"你刮瓶盖刮出谢了，还会继续刮吗"——段永平后来用这个比喻解释离开原因。 | 二手（访谈记录） |
| **1995年8月28日** | 向小霸王递交辞呈，承诺一年内不生产同类产品。带走陈明永、沈炜、金志江等6位核心高管。 | 二手（多个来源一致） |
| **1995年9月18日** | 在东莞创立步步高电子有限公司。陈明永、沈炜等骨干随他一起创业。实行股份制改革，几乎所有中层管理人员都入了股。 | 二手（百科、多个报道一致，具体日期存在版本差异） |

## 三、步步高创业与分拆（1995-2001）

| 时间 | 事件 | 信息来源 |
|------|------|----------|
| **1995年9月** | 创立步步高，提出"敢为天下后"策略——不害怕跟在别人后面进入市场，但要做到更好。 | 二手（多个报道） |
| **1996年底** | 步步高中标央视黄金时段广告。 | 二手（百科） |
| **1997年** | 步步高以1.59亿元拿下次年春晚零点报时赞助商。进入VCD市场，采用"后发优势"策略，在已有200多家厂商的市场中脱颖而出。 | 二手（百科） |
| **1997-1998年** | 步步高电话机、VCD等产品迅速成为市场领导品牌。无绳电话"喂~小丽呀"广告深入人心。 | 二手（媒体报道） |
| **1998年** | 与刘昕相识。刘昕为中国人民大学新闻系校友（1986-1990就读），当时在美国《棕榈滩邮报》任首席摄影记者，回国探亲时与段永平相识。 | 二手（陈露露等资料整理） |
| **1998年** | 与刘昕相识仅两个月后闪婚。刘昕提出移居美国的要求，段永平答应。 | 二手（多个报道） |
| **1999年** | 将步步高业务拆分为三家独立公司：步步高教育电子（金志江负责）、步步高通讯科技（沈炜负责，后成为vivo）、步步高视听电子（陈明永负责，后成为OPPO）。 | 二手（百科、多个报道一致） |
| **1999年分拆时** | 段永平主动将自己70%的股份削减至OPPO约10%、vivo不到20%，剩余分给核心团队和员工。这种"共享共治"模式后来成为步步高系成功的关键。 | 二手（搜狐、今日头条） |
| **2000年** | 步步高系企业基本完成分拆架构，段永平开始淡出日常管理。 | 二手（时间线整理） |
| **2001年** | 步步高拆分后不久，三家步步高公司共同出资注册OPPO品牌。陈明永后来买断OPPO品牌股权。 | 二手（雪球、媒体报道） |
| **2001年** | 段永平正式移居美国，退居幕后，以投资人身份控制步步高系。对三位接棒者的临别赠言："如果实在干不好，就体面地把企业关掉。不要指望我回来，这是你们的事了。" | 二手（多个报道一致） |

## 四、移民美国与投资生涯开启（2001-2010）

| 时间 | 事件 | 信息来源 |
|------|------|----------|
| **2001年** | 为与妻子刘昕团聚，定居美国加州硅谷帕洛阿尔托。从企业家转型投资人。 | 二手（多个报道一致） |
| **2001年底** | 开始关注网易股票。当时网易因互联网泡沫破灭、财务丑闻和退市风险，股价从17美元跌至不足1美元。 | 二手（今日头条、雪球分析） |
| **2002年初** | 与丁磊（网易创始人）交流，丁磊决定转型游戏业务，做游戏机出身的段永平认可这一方向。聘请律师评估网易法律风险，认为摘牌概率低。 | 二手（多个报道一致） |
| **2002年4月** | 开始重仓买入网易股票。从2002年2月至4月，以约1美元/股均价买入152万股，后增持至205万股（占网易总股本6.8%），总投资约200万美元。成为网易第二大个人股东。 | 二手（雪球、CSDN等多个来源） |
| **2002-2003年** | 网易股价从约1美元涨至70美元以上（2003年10月），段永平一战成名，被称为"中国巴菲特"。 | 二手（多个报道一致） |
| **约2003-2010年** | 持有网易股票约8-9年，在30-35美元（拆股后）分批卖出，总回报约100-140倍。 | 二手（雪球分析） |
| **2003年** | 开始建仓贵州茅台。受1998年山西假酒案和2003年非典疫情影响，白酒行业低迷，茅台股价被压制。段永平认为茅台具有"品牌垄断、定价权、抗通胀"三重优势。 | 二手（雪球分析） |
| **2005年** | 与妻子刘昕在美国成立家庭慈善基金Enlight Foundation，专注教育捐赠。 | 二手（百科） |
| **2006年6月** | 以62.01万美元在eBay上拍得与巴菲特共进午餐的机会，成为首位与股神共餐的华人。 | 一手/二手（段永平原话和媒体报道） |
| **2006年** | 与丁磊联合向浙江大学捐赠4000万美元。其中段永平捐赠3000万美元，丁磊捐赠1000万美元。这是当时中国高校接收的最大单笔捐款。 | 二手（多个报道一致） |
| **2007年5月9日** | 在纽约与巴菲特共进午餐（3小时）。段永平带黄峥（当时26岁）一同参加。核心收获："买股票就是买公司"这一句话就够了。巴菲特回复："这也是我当年跟格雷厄姆学到的最重要的一句话。" | 一手/二手（访谈记录） |
| **2008年5月14日** | 汶川地震，段永平捐款300万美元。 | 二手（百科） |
| **2008年** | 在中国民政部门注册成立心平公益基金会，主要用于教育捐赠。刘昕为主要发起人。 | 二手（百科） |
| **2010年** | 与妻子刘昕向中国人民大学捐赠3000万美元，设立教育基金。这是人大历史上获赠金额最大的捐款。 | 二手（百科、多个报道一致） |

## 五、OPPO/vivo成长与成熟投资（2011-2020）

| 时间 | 事件 | 信息来源 |
|------|------|----------|
| **2008-2011年** | 步步高通讯科技（沈炜负责）从功能机向智能机转型。2011年正式推出vivo品牌。OPPO（陈明永负责）从视听产品向手机转型。段永平作为投资人角色，不参与日常管理。 | 二手（多个报道） |
| **2011年** | 向北京师范大学捐赠设立"北京师范大学永平自立贷学金"。 | 二手（北师大官方） |
| **2011年11月** | 开始买入苹果股票。当时乔布斯8月辞任CEO，10月5日去世，苹果股价从高位跌30%，市场质疑"没有乔布斯的苹果还是苹果吗"。段永平认为苹果的平台已搭好，库克可以胜任。首次买入价约13.75美元/股（前复权）。 | 一手/二手（段永平晒单、多个报道） |
| **2011年起** | 长期重仓苹果，持仓比例一度达80%-90%。段永平向巴菲特解释苹果价值："苹果的生意模式就像一个收费站"——巴菲特一句话就明白了。 | 一手/二手（访谈记录） |
| **2012年** | 塑化剂事件冲击白酒行业，茅台股价暴跌。段永平认为这是加仓良机，持续买入并重仓茅台。 | 二手（雪球分析） |
| **2013-2014年** | 白酒行业深度调整，"八项规定"和"禁酒令"打压高端消费。段永平继续加仓茅台，认为市场恐慌提供了极佳买入机会。 | 二手（雪球分析） |
| **2015年** | 以网名"大道无形我有型"入驻雪球平台。原ID"大道无形"已被注册，改为"大道无形我有型"。此后在雪球、网易博客等平台分享投资心得。 | 一手/二手（段永平自述） |
| **2015年至今** | 在雪球平台持续活跃，分享投资理念、持仓信息和与网友互动，成为中国价值投资领域的意见领袖。 | 一手（雪球发言） |
| **约2015-2020年** | 投资、加持腾讯等港股标的。布局拼多多（风险投资性质），是黄峥早期投资者和导师。 | 二手（持仓披露、采访记录） |

## 六、公开活动与捐赠高峰（2020-2025）

| 时间 | 事件 | 信息来源 |
|------|------|----------|
| **2020年前后** | OPPO、vivo成为中国手机市场头部品牌。2025年一季度，vivo以17.4%市场份额连续四年成为中国手机销量第一。段永平虽不参与管理，但其"本分文化"和"共享共治"模式被认为是步步高系成功的关键。 | 二手（市场数据、报道） |
| **2022年** | 在浙江大学125周年校庆之际，向母校捐赠1.7亿元。 | 一手（浙大官方） |
| **2022年12月31日** | 与浙江大学教育基金会签署捐赠协议，再次捐赠1亿元人民币。 | 二手（媒体报道） |
| **2023年12月31日** | 向浙江大学教育基金会进行大额捐赠，合计超10亿元，用于学校基础设施建设、人才培养、学科建设、师生文体活动等。累计向浙大捐赠超过15亿元，成为浙大最捐校友之一。 | 二手（媒体报道） |
| **2024年** | 向中欧教育发展基金会捐赠1亿元证券。中欧商学院上海校区将第四教学中心命名为"段永平教学中心"。 | 二手（雪球报道） |
| **2025年1月5日** | 重返母校浙江大学，与师生进行约90分钟交流。这是他"退休"二十多年后的罕见公开演讲。核心观点：做对的事情，把事情做对；不要想赚快钱；想5年、10年、20年以后的决策对不对。 | 一手（浙大演讲实录） |
| **2025年3月11日** | 在雪球发文"哈，终于来了！很少享受到空仓的快感，投资以来第一次"。引发市场对其做空美股的质疑。 | 一手（雪球发言） |
| **2025年4月** | 宣布"会有相当长一段时间不再上雪球"，原因是"语言环境不好"。 | 一手（雪球发言） |
| **2025年8月** | 对泡泡玛特表态："依然无法理解人们为什么会需要这个东西""担心过两年市场需求消失"，称不会投资泡泡玛特。 | 一手（雪球发言） |
| **2025年10月16日** | 在美国加州接受雪球创始人方三文（方丈）深度访谈，时长93分钟。这是段永平"退休"后的罕见公开对话。 | 一手（访谈实录） |
| **2025年10月13日** | 在社交平台表示"今天买了点茅台"。 | 一手（雪球发言） |
| **2025年11月11日** | 访谈内容发布。同日与北京师范大学教育基金会签署协议，捐赠2.2亿元人民币，用于理工综合体建设。北师大将珠海校区理工综合体学术云环冠名为"段永平学术云环"，图书馆冠名为"彭建华图书馆"（以其母亲命名）。 | 一手/二手（北师大官方） |
| **2025年11月** | 恢复雪球活跃，深度访谈发布后引发广泛关注。 | 一手（雪球活跃） |
| **2025年12月** | 接受王石、田朴珺夫妇访谈（"共同说"），谈本分文化、步步高分拆决策、长期主义等。访谈中透露vivo连续四年成为中国手机销量第一。 | 一手（访谈内容） |
| **2025年底** | 据不完全统计，过去二十年向中国教育事业累计捐款总额接近20亿元。 | 二手（媒体报道） |

## 七、2026年最新动态

| 时间 | 事件 | 信息来源 |
|------|------|----------|
| **2026年1月** | 调仓换股：卖出中国神华，重仓茅台。1月21日挂单1365元/股买入2万股茅台。1月27日以1330元左右加仓约7.72万股茅台（投入约1亿元）。短短5天内浮盈超1000万元。 | 一手（雪球发言） |
| **2026年1月** | 晒出2011年11月买入苹果账户，浮盈1881.8%（约14年16倍），年化收益约24%。苹果持仓占投资组合约60%，市值约88.69亿美元。 | 一手（雪球晒单） |
| **2026年3月** | 对泡泡玛特态度大反转。称已决定收回此前"不投资泡泡玛特"的说法。原因：阅读《因为独特》（王宁深度访谈）、实地调研海外门店、研究财报（2025年营收371.2亿元，同比增184.7%）。 | 一手（雪球发言） |
| **2026年3月底** | 因搞错港股期权规则，一口气卖出大量泡泡玛特看跌期权（put），差点需花68亿港元接货成为第三大股东。称这是"乌龙操作"。后称"我现在确实是王宁的粉丝""Right business, Right people! Right price seems to appear too." | 一手（雪球发言） |
| **2026年4月** | 据H&H International Investment持仓披露，段永平仍重仓苹果（约60%）、伯克希尔（约18%）、拼多多等。过去一年减持苹果26%（从80%降至60%），但仍为第一大持仓。 | 二手（持仓披露） |
