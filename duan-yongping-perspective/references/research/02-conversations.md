## 一、深度访谈

### 1.1 2025年雪球《方略》访谈（核心素材）

**来源**：雪球《方略》第三季第一期
**时间**：2025年10月16日（访谈） / 2025年11月11日（发布）
**对谈者**：方三文（雪球创始人）
**时长**：93分钟
**可信度**：⭐⭐⭐⭐⭐（一手原话，视频记录）

#### 访谈背景
这是段永平"退休"二十多年后罕见的公开视频对话。在加州面对面进行，信息量巨大，很多内容首次公开详细披露。

#### 核心数据
- 提到「犯错」10次
- 提到「看懂」12次
- 提到「很难」29次
- 这些词频反映了他一以贯之的思考方式

#### 即兴表达特点

**1. 面对无厘头问题的回应方式**
```
方三文：你是如何度过平常的一天的？比如今天。
段永平：打球，运动。
方三文：天天如此吗？
段永平：差不多吧，也没有什么，就是不同的球场打球。
方三文：唯一变化的是这个球场的地点，还有打球的伙伴？
段永平：伙伴有时候有，有时候没有。
方三文：明白了，这就是你的平淡无奇的一天。
段永平：挺好。这没有什么平淡无奇，这就正常。
```
- **特点**：不被"平淡"的定义牵引，直接用自己的标准回应
- **思维方式**：拒绝被外部评价框架绑架，坚持自己的坐标系

**2. 被追问"成长环境影响"的回应**
```
方三文：小时候是在什么样的环境下长大的，这个成长的环境跟你现在的性格、成就有没有什么关系？
段永平：这个就不知道了。我是在南昌出生，六岁左右跟父母下放到江西安福县...
```
- **特点**：承认"不知道"因果关系，只讲事实
- **思维方式**：区分事实与归因，不强行建立因果联系

**3. 关于"安全感与神经质"的即兴分析**
```
段永平：我爸爸妈妈好像不太怎么管我，所以我小时候就对我们很好，但是呢，没有太多的要求...所以我觉得我还是比较有安全感的，就不是说怕这个怕那个。
```
- **即兴洞察**：安全感与理性的关系——"没有安全感的人很难理性"
- **底层逻辑**：童年安全感 → 敢于做决策 → 理性决策能力

**4. 关于"胸无大志"的自我描述**
```
段永平：我一直胸无大志，觉得自己就是普通人，能过好小日子、做喜欢的事就挺开心，没想着非要成就点什么。
```
- **特点**：坦然承认"胸无大志"，反常规叙事
- **深层含义**：真正的"志"不需要刻意追求外部成就

#### 被追问时的回答模式

**模式1：回到常识**
```
问：茅台真的涨价了，你真是神人啊！
段：这是个简单逻辑题啊，有什么神的？茅台其实并没有别的选择了...
```
- 把"神预测"拆解为"简单逻辑"
- 强调"知道什么会发生"比"知道什么时候会发生"容易得多

**模式2：承认边界**
```
问：[关于拼多多能否持续]
段：我觉得大概率他们应该活得还不错，但是不是100%，我觉得苹果我可以说我很肯定的知道苹果会活得很好，拼多多我没有那么清晰...
```
- 明确区分"确定性程度"：苹果vs拼多多
- 不强装全知，显式标注置信度

**模式3：拒绝回答/承认不懂**
```
问：[如何度过平常一天]
段：打球，运动...挺好。这没有什么平淡无奇，这就正常。
```
- 不接受问题预设的框架（"平淡无奇"=bad）
- 重新定义问题的坐标系

#### 改变立场/认知进化的案例

**案例1：对万科、创维的反思**
```
段永平原话（2018年9月30日，雪球）：
"今天再看当年买万科和创维的决策，显然是不对的。这点从对'现金流折现'的理解上就能看出来。"

"当时觉得这些公司便宜，其实便宜不便宜和未来现金流有关，和过去的东西没直接关系。比如，当时觉得万科的净资产很高（100多亿？），觉得很便宜。其实，如果不能产生自由现金流，这些净资产不仅不值钱，还会产生负的现金流。"
```
- **认知跃迁**：从"净资产/账面价值"→"未来现金流折现"
- **反思深度**：承认早期成功有运气成分，投资体系从"价格"转向"价值"

**案例2：卖Yahoo买Apple的转折**
```
完整语境：早期买卖Yahoo → 后来发现苹果才是真正的好生意 → 承认当时没看透
```
- **关键洞察**：投资认知的迭代，从"便宜"到"好商业模式"

## 二、论坛问答（雪球互动）

### 2.1 雪球互动规则

**来源**：段永平雪球公告
**时间**：2018年
**可信度**：⭐⭐⭐⭐⭐（一手原话）

```
段永平公告：
简单讲，原则和原来一样，大致就是：
1. 一天一条
2. 不要给我提建议
3. 我在这里分享我的想法，但不想和任何人讨论任何问题。大家有问题尽管问，我也会尽能力回答，但希望问题简单清晰。我不希望看到那种连自己都搞不清楚想问什么的问题哈。

在雪球加关注和在网易加博友的规矩一样，需要两个人的推荐，我也会看下他们过去都说过啥，最后才决定是不是要加。
```

**特点**：
- 控制互动边界，不被社区绑架
- 要求提问者先想清楚问题
- 不接受"建议"，只分享不讨论

### 2.2 经典问答模式（雪球）

**模式1：简洁回应+点到为止**

```
用户：段总，大A上市的除了茅台，还有哪些公司值得长期持有哇？
段永平：你问我，我问谁？
```
- **特点**：用反问打掉"抄作业"心态
- **深层含义**：投资需要自己思考，不能靠别人推荐

**模式2：承认不懂+点出关键**

```
段：我确实是个运气很好的普通人，悟到了一些偷懒的办法，一些或许对其他普通人也有点帮助的办法。做正确的事情会少折腾很多。
```

**模式3：用类比和常识回应**

```
段：很多事情知道什么会发生要比知道什么时候会发生要容易的多的多。
```

## 四、失败案例与反思

### 4.1 做空百度（最大失败）

**来源**：段永平雪球自述
**可信度**：⭐⭐⭐⭐⭐（一手原话）

```
段永平原话：
呵呵，亏钱的投资有好几次了，多数都可以原谅，但有一次属于极度愚蠢，就是跑去short百度。之前投资表现非常好，有点飘飘然，真以为自己很厉害。

开始还是想小玩玩，后来又不服输，最后被夹空而投降，所有账号加起来亏了很多钱，大概有1.5-2亿美金，其中有个账号到现在都翻不了身。

最可惜的是，这次错误把我们的现金储备全部消耗掉了，机会成本巨大，不然这次金融危机我就可以帮大家赚更多钱了。

而且最遗憾的是，这个错误是发生在巴菲特叫我不要做空之后。现在知道什么叫不听老人言的后果了吧？
```

**反思要点**：
1. 违背"不做空"原则
2. 投机心态驱动（飘飘然）
3. 不服输→被夹空
4. 机会成本巨大（错过金融危机机会）

### 4.2 天然气指数UNG

**来源**：雪球问答
**可信度**：⭐⭐⭐⭐（一手原话）

```
段永平：
亏了超百万美元，止损后避免后续4倍亏损。

错误根源：
1. 违背"不懂不做"原则：误判UNG与天然气价格的线性关系，未理解金融衍生品的时间损耗机制（Contango效应）
2. 轻信表面数据：依赖历史价格走势而非底层资产逻辑
```

**教训**：
- 发现错误要尽快改，不管多大的代价都可能是最小的代价

### 4.3 早期投资万科、创维的反思

```
段永平（2018年反思）：
今天再看当年买万科和创维的决策，显然是不对的。

当时觉得这些公司便宜，其实便宜不便宜和未来现金流有关，和过去的东西没直接关系。
```

## 十、关键发现

### 10.1 即兴回答的三个层次
1. **承认不懂**：直接说出来，不装
2. **回到常识**：把复杂问题拆解为简单逻辑
3. **标注置信度**：区分"确定"、"大概率"、"不懂"

### 10.2 面对追问的策略
- 不接受问题的天然框架
- 用反问打掉"抄作业"心态
- 用自己的坐标系重新定义问题

### 10.3 改变立场的坦诚度
- 主动承认早期错误（万科、创维）
- 公开失败案例并总结教训
- 不装全知新，显式标注认知边界
