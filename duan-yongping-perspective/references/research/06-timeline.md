## 一、早年经历（1961-1988）

| 时间 | 事件 | 对思维的影响 | 来源 |
|------|------|-------------|------|
| 1961年3月10日 | 出生于江西省南昌市（原籍江西泰和县），父母均为江西水利电力学院教师（大学教师） | 知识分子家庭背景，奠定教育重视程度；童年家境相对优越 | 百度百科、豆瓣、新浪财经 |
| 约1967年（6岁） | 随父母下放到江西安福县农村（"上山下乡"运动） | 体验艰苦生活，培养"本分"朴实性格；童年吃过苦，形成安全感与平常心 | 2025年雪球访谈 |
| 1970年代 | 在安福县农村多次搬家（共搬5-6次），最终搬回南昌附近石岗 | 适应变化的能力，漂泊经历塑造坚韧性格 | 2025年雪球访谈 |
| 1977年10月 | 国家恢复高考；首次参加高考，仅考80多分（满分500分），落榜 | 认识到自己并非天赋异禀，需要努力才能成功；复读决心坚定 | 知乎年谱、今日头条 |
| 1978年7月 | 复读一年后，以400多分考入浙江大学无线电系（恢复高考后首届大学生，全校唯一本科生） | 坚持努力的价值；技术背景为日后创业打下基础 | 百度百科、知乎、网易 |
| 1978-1983年 | 浙江大学无线电系就读（中间曾休学一年） | 专业知识积累；与母校建立深厚情感纽带（日后巨额捐赠）；大学报到提着从舅舅家借来的17寸黑白电视 | 百度百科、今日头条 |
| 1982年 | 浙江大学毕业，被分配到无锡湖光仪器厂；后与一上海同学交换，去了北京电子管厂（国企，月薪46元） | 进入体制内；体验"铁饭碗"但感到迷惘，萌生创业念头 | 百度百科、豆瓣、今日头条 |
| 1983-1986年 | 在北京电子管厂工作3年，工资从46元涨到第二年的56元，到1986年涨到80多元/月 | 对国企体制产生质疑；开始思考人生方向；"铁饭碗端了3年就觉得厌倦了" | 今日头条时间线 |
| 1986年 | 考入中国人民大学经济学院，攻读计量经济学研究生 | 经济学思维启蒙；为未来投资奠定理论基础；认识后来的妻子刘昕 | 百度百科、新浪财经 |
| 1988年7月 | 人民大学研究生毕业（未提交学位论文，最高学历为学士）；第一次商业尝试——贩卖章光101生发剂（未赚大钱） | **思想转折点1**：商业兴趣被激发；拒绝请人代写论文体现"本分"诚信原则 | 今日头条、知乎 |

## 二、创业历程（1989-2001）

| 时间 | 事件 | 对思维的影响 | 来源 |
|------|------|-------------|------|
| 1988年 | 研究生毕业后，有机会留在北京但选择南下广东发展；先进入一家高学历公司，因不适应很快离开 | 选择机会多于安稳；对不尊重人的管理方式反感 | 今日头条 |
| 1989年3月 | 经同学介绍，进入中山市怡华集团下属的"日华电子"小工厂（亏损200万元）任厂长 | **思想转折点2**：从技术/理论转向实业经营；选择"从0到1"的挑战 | 百度百科、搜狐 |
| 1989-1990年 | 生产家用电视游戏机（模仿任天堂红白机），聚焦游戏机市场 | "敢为天下后"策略萌芽：模仿+改良，快速切入市场 | 搜狐 |
| 1990年 | 将工厂更名为"小霸王电子工业公司"，正式创立"小霸王"品牌 | 确立品牌战略；改名象征新起点 | 百度百科 |
| 1991年 | 推出小霸王学习机：将键盘与游戏手柄结合，主打"学习编程、练习打字" | 精准捕捉"望子成龙"社会心理；产品创新体现用户洞察 | 网易、今日头条 |
| 1991-1992年 | 砸下40万元拍下央视黄金时段广告，请成龙代言 | 品牌营销能力达到新高度；"拥有一台小霸王，打出一个万元户"广告语家喻户晓 | 百家号、今日头条 |
| 1991-1994年 | 小霸王销售额从0飙升至年产值10亿元，市场份额近80%，净利润突破8000万元 | 商业成功验证方法；成为行业第一品牌；1994年市场调查显示小霸王是中国人最熟悉的电脑品牌（超过联想和IBM） | 网易、搜狐、今日头条 |
| 1992-1994年 | 多次向集团提出股份制改革方案：初始承诺"三七开"（段30%），后改为"二八开"（段20%），最终"一九开"（段10%）甚至更低 | 认识到激励机制对企业发展的重要性；无法兑现对团队的承诺感到愧疚 | 今日头条、雪球、网易 |
| 1995年8月28日 | 正式离开小霸王，提出辞职信；集团老板陈健仁多次拒绝股份制改革方案 | **思想转折点3**：决定离开自主创业；"本分"理念：宁愿重新开始也不失信于团队 | 今日头条、知乎专栏 |
| 1995年9月 | 带领陈明永、沈炜等六位核心骨干，在东莞长安镇创立步步高电子有限公司；签订竞业协议，不能做游戏机 | "本分"体现：不拿原公司一分钱、不挖人才；重新开始 | 博客园、网易、微博 |
| 1995年底 | 步步高以VCD和无绳电话切入市场，邀请李连杰代言，在央视投放大规模广告 | 产品策略调整：避开竞业协议领域；营销打法复制小霸王经验 | 微博、搜狐 |
| 1996年底 | 步步高中标央视黄金时段广告；1999-2000年光新闻联播标版就砸8000万元 | 品牌营销能力达到巅峰；"敢为天下后，后中争先"策略成功 | 微博、百度百科 |
| 1997年 | 导入CIS企业形象识别系统，品牌形象全面提升 | 系统化品牌建设意识 | 今日头条 |
| 1998-1999年 | 步步高成为VCD、无绳电话市场领导品牌；业务覆盖教育电子、视听、通讯三大板块 | 跟随策略成功；多元化布局形成 | 百度百科 |
| 1999年 | **关键决策**：将步步高业务拆分为三——教育电子（黄一禾→金志江）、视听电子/OPPO（陈明永）、通信科技/vivo（沈炜），股权独立、互无从属 | **思想转折点4**：去中心化授权模式；"无为而治"的企业治理哲学；信任传人胜过控制；自己只各留10%股份 | 百度百科、网易、博客园、搜狐 |
| 2000年 | OPPO年销5.2亿元，vivo跻身行业前三；步步高系开始崛起 | 拆分决策被证明正确；"本分"文化在三家公司的传承 | 百家号、雪球 |
| 2001年 | 移居美国加州，正式"退居二线"，为兑现陪妻子的承诺 | **思想转折点5**：从实业家转型投资人；家庭优先的价值观；"胸无大志"的平常心 | 搜狐、新浪、雪球访谈 |
| 2001年 | 步步高面临功能手机库存危机，几十亿积压；段永平准备在美炒股现金作为破产兜底，对团队说"做不好就好好关掉，但绝不能欠供应商一分钱" | "本分"底线思维；对供应链负责的态度 | 百家号 |
| 2001年 | 将步步高三家公司股权交给黄一禾（教育）、陈明永（OPPO）、沈炜（vivo）管理，自己只担任顾问、保留小股份 | 彻底的授权理念；"不控股但培养人"的管理哲学 | 雪球 |

## 三、投资转型（2001-现在）

| 时间 | 事件 | 对思维的影响 | 来源 |
|------|------|-------------|------|
| 2001年初 | 移居美国加州，开始研究巴菲特，接触价值投资理念；阅读巴菲特历年致股东信 | **思想转折点6**：发现投资哲学——"买股票就是买公司"；从实业家转型投资人 | 雪球访谈、知乎 |
| 2001年中期 | 丁磊因网易股价暴跌（从15.5美元跌至0.65美元）、财务造假危机、退市威胁求助段永平；两人在东莞会面，段永平鼓励丁磊转型游戏 | 建立"买股票就是买公司"的投资框架；"你现在不就有一家公司吗"改变了丁磊命运 | 搜狐、雪球、新浪财经 |
| 2001年7月-2002年4月 | 段永平通过美国律师、会计师了解网易财务造假真相后，从约0.66-1美元/股开始建仓网易，最终持有约205万股（占6.8%）；投入约200万美元（自有100万+借款100万） | **思想转折点7**：逆向投资能力显现；"股价低于净现金"的极度低估判断；风险投资式的勇气 | 雪球、知乎、今日头条 |
| 2002年 | 网易转型游戏成功，《梦幻西游》上线，股价开始反弹；6个月内网易暴涨20倍 | 投资眼光被验证；但段永平选择继续持有 | 雪球 |
| 2002-2010年 | 持有网易8-9年，最终以约120-130美元/股退出，百倍回报（约100-200倍） | 长期持有理念确立；"买股票就是买公司"的真正实践；"那些钱对我来讲并不多，所以我可以比较理性地去想一件事情" | 雪球、投资界 |
| 2003年 | 投资"美国拖车租赁公司"（U-Haul相关），3-5美元买入，80-100美元卖出，约20倍回报 | 投资体系成熟：低估+催化剂 | 雪球 |
| 2003年 | 《福布斯》中国富豪榜第71位，资产约10亿人民币 | 商业成就被认可 | EDP培训网 |
| 2004年 | 投资创维数码，市值20亿港币时买入，约1块多买入，5-8块卖出，约5倍回报 | AH股套利能力体现 | 雪球 |
| 2004年 | 黄峥从威斯康星大学毕业，段永平力荐其加入谷歌而非微软 | 对年轻人的培养和眼光；黄峥后来的成功证明判断 | 知乎、雪球 |
| 2005年 | 投资万科A股，市值150亿时买入，约2块多买入，10-15块卖出，约6倍回报 | 中国地产黄金时代的认知 | 雪球 |
| 2006年6月30日 | 以62.01万美元拍下巴菲特午餐（首位华人），次年5月在纽约Smith & Wollensky餐厅赴宴；带26岁黄峥同行 | 投资理念得到偶像确认；"买股票就是买公司"理念坚定；慈善观形成；黄峥只问了一个问题"黄金为什么值钱" | 新浪财经、搜狐、澎湃新闻、今日头条 |
| 2006年 | 与妻子刘昕成立Enlight Foundation慈善基金会 | 财富观转变：钱是工具不是目的 | 搜狐 |
| 2008年 | 投资通用电气（GE），约6-9美元买入，20多美元卖出，约2.5倍回报 | 蓝筹股投资能力 | 雪球 |
| 2009年 | 投资雅虎，目的是淘宝和阿里巴巴集团的价值，赚1-2倍 | **投资反思点**：雅虎让段永平意识到"不能只看资产，要看生意是否可持续" | 雪球 |
