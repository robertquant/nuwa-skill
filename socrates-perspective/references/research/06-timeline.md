# 苏格拉底(Socrates)生平时间线

> **说明**：所有年份为公元前(BCE)。苏格拉底本人未留下任何著作，主要史料来源于柏拉图(Plato)、色诺芬(Xenophon)、阿里斯托芬(Aristophanes)及亚里士多德(Aristotle)。不同来源存在记载差异，本文注明主要争议点。

---

## 生平时间线

| 年份 | 事件 | 意义 |
|------|------|------|
| **约469-470** | 苏格拉底出生于雅典阿洛佩凯(Alopece) | 父亲索弗洛尼斯库斯(Sophroniscus)为石匠/雕刻师；母亲菲娜蕾特(Phaenarete)为助产士。家庭属平民阶层，非贵族。来源：柏拉图《申辩篇》20d、色诺芬《回忆录》|
| **约451-450** | 阿尔西比亚德斯(Alcibiades)出生 | 后成为苏格拉底最著名的学生与挚友之一，也是苏格拉底政治麻烦的重要来源 |
| **约440年代** | 接受常规希腊式教育 | 学习音乐、体育、诗歌、读写能力。来源：柏拉图、色诺芬著作 |
| **约430年代** | 从事雕刻工作 | 传曾创作雅典卫城附近的"美惠三女神"(Graces)雕像（2世纪仍有记载）。来源：后世传记传统（非当代文献） |
| **432** | 参加波提狄亚战役(Battle of Potidaea) | 作为重装步兵参战。据柏拉图《会饮篇》记载，苏格拉底在战场上表现出非凡耐寒能力，并救了阿尔西比亚德斯的性命。来源：柏拉图《会饮篇》219e-221b、《卡尔米德篇》153a-d |
| **约424** | 参加德里昂战役(Battle of Delium) | 雅典惨败。苏格拉底作为7000重装步兵之一参战。柏拉图记载他在撤退中表现出冷静沉着。拉尔凯斯(Laches)与其并肩作战后赞誉其勇气。来源：柏拉图《拉凯斯篇》181b、《会饮篇》221a-b |
| **423** | 阿里斯托芬喜剧《云》首演 | 将苏格拉底讽刺为研究自然哲学的"思想铺"主持人，教授青年歪理邪说。苏格拉底后来在审判中称此剧毒害了公众对他的印象。来源：阿里斯托芬《云》、柏拉图《申辩篇》18a-19d |
| **422** | 参加安菲波利斯战役(Battle of Amphipolis) | 雅典再次战败。此为苏格拉底最后一次有记载的军事服役。来源：柏拉图《申辩篇》28e、色诺芬 |
| **约416左右** | 德尔斐神谕事件 | 朋友凯勒丰(Chaerephon)赴德尔斐神庙询问：是否有人比苏格拉底更智慧？神谕回答"没有"。苏格拉底困惑于此，开始四处询问自认有智慧者（政治家、诗人、工匠），发现他们虽自称有知实则无知，从而领悟神谕真意——"我知我无知"。来源：柏拉图《申辩篇》20e-23b（具体年份无确切记载，推测为中年） |
| **约415** | 赫尔墨斯神像毁坏案(Mutilation of the Herms) | 雅典出土征西西里前夕，城内赫尔墨斯神像遭破坏。阿尔西比亚德斯等人被指控，阿尔西比亚德斯逃往斯巴达。此事件使苏格拉底因其与阿尔西比亚德斯的密切关系受到牵连。来源：修昔底德、普鲁塔克 |
| **约411** | 雅典寡头政变（四百人议事会） | 阿尔西比亚德斯参与策划。此后雅典政局动荡。 |
| **约404** | 伯罗奔尼撒战争结束，雅典战败 | 斯巴达扶植"三十僭主"(The Thirty Tyrants)执政，首领克里提亚斯(Critias)和卡尔米德(Charmides)均为苏格拉底前学生/圈中人。来源：色诺芬《希腊史》、柏拉图 |
| **404-403** | 三十僭主统治时期 | 克里提亚斯执政期间暴虐无道。据记载，三十僭主曾命令苏格拉底协助逮捕民主派将领莱昂(Leon)，苏格拉底拒绝从命。同时克里提亚斯禁止苏格拉底与30岁以下青年对话。来源：柏拉图《申辩篇》32c-e、色诺芬《回忆录》I.2.31-38 |
| **403** | 民主政体恢复，大赦颁布 | 流亡民主派推翻三十僭主，恢复民主。克里提亚斯、卡尔米德均战死。新政权颁布大赦令，禁止因政治原因提起诉讼。这解释了为何对苏格拉底的指控以宗教罪名而非政治罪名出现。 |
| **约402-403左右** | 与美诺、安尼图斯对话 | 柏拉图《美诺篇》中记载苏格拉底与安尼图斯对话，后者后来成为苏格拉底主要控告人之一。安尼图斯指责诡辩家败坏青年。来源：柏拉图《美诺篇》89e-95a |
| **399** | 被起诉 | 控告人：美勒图斯(Meletus，诗人)、安尼图斯(Anytus，皮匠、民主派领袖)、吕孔(Lycon，演说家)。指控：①不敬城邦所信之神；②引进新神灵；③败坏青年。来源：柏拉图《申辩篇》24b、色诺芬《申辩篇》10-12、第欧根尼·拉尔修《名哲言行录》 |
| **399** | 审判与申辩 | 雅典法庭由501人陪审团审理。苏格拉底拒绝聘请演说家撰写辩护词，亲自申辩。他拒绝乞怜，宣称自己是雅典的"牛虻"，为神所遣以警醒城邦。拒绝认罪，反提议应在普吕塔内翁(Prytaneum)享受公费款待（英雄待遇）。来源：柏拉图《申辩篇》 |
| **399** | 被判处死刑 | 第一轮投票：有罪（约280票对221票）。按雅典惯例被告可提议较轻惩罚，苏格拉底最初提议仅罚一明那(Mina)，后按柏拉图等人建议提议30明那罚金，但拒绝流放或停止哲学活动。最终陪审团以更大比例判决死刑。来源：柏拉图《申辩篇》36a-38b |
| **399** | 入狱等待执行 | 因神圣船只(Delian mission)出航，行刑延后约一个月。期间朋友（克里托Criton等）安排越狱，苏格拉底拒绝。来源：柏拉图《克里托篇》 |
| **399** | 狱中对话 | 柏拉图《斐多篇》记载了苏格拉底临终前与朋友讨论灵魂不朽的对话。在场的有克里托、阿波罗多洛斯、西米亚斯、刻贝斯等。来源：柏拉图《斐多篇》59c-63e |
| **399** | 饮毒芹汁(hemlock)去世 | 年约70岁。据《斐多篇》描写，行刑者递给苏格拉底毒药，他从容饮下，最后话语涉及欠阿斯克勒庇俄斯(Asclepius)一只鸡的奉献。来源：柏拉图《斐多篇》117a-118a |

## 关键人物的争议与史料问题

### 1. 历史真实性问题（苏格拉底问题）
- 苏格拉底本人未留下任何文字，我们对其了解完全依赖他人记载
- **柏拉图**：主要来源，但学术界对其"苏格拉底"是历史人物还是哲学代言人有争议
- **色诺芬**：提供较务实、日常的苏格拉底形象，但有时与柏拉图记载矛盾
- **阿里斯托芬**：最早来源（《云》首演于公元前423年），但其为喜剧讽刺，史料价值存疑

### 2. 德尔斐神谕日期争议
- 柏拉图《申辩篇》未给出具体日期
- 学者推测应在苏格拉底中年（约40-50岁）
- 与其"哲学转向"的时间相关

### 3. 家庭背景争议
- 婚姻：有文献（第欧根尼·拉尔修）提及苏格拉底可能有两任妻子（桑提珀Xanthippe与米尔托Myrto），甚至同时娶两人（因雅典男性不足）
- 子女：有三个儿子：兰普罗克勒斯(Lamprocles)、索弗洛尼斯库斯(Sophroniscus)、墨涅克塞努斯(Menexenus)。柏拉图《斐多篇》记载苏格拉底去世时幼子尚小

### 4. 三十僭主时期立场争议
- 苏格拉底并未逃离雅典（许多民主派人士流亡）
- 其弟子克里提亚斯是三十僭主首领
- 但苏格拉底拒绝执行逮捕民主派将领莱昂的命令
- 民主恢复后，这一"双面"立场使其成为政治清算目标

### 5. 死因的政治背景
- 公元前399年距离雅典战败（404）仅五年，距离三十僭主倒台（403）仅四年
- 大赦令禁止政治指控，故以宗教罪名起诉
- 控告人安尼图斯是推翻三十僭主的民主流亡派领袖
