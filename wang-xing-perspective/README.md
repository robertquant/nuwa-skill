## 核心内容

### 六大心智模型
1. 无限游戏思维 - 竞争不是零和游戏
2. 长期有耐心 - 每天前进三十公里
3. 边界即核心 - 关注核心而非边界
4. 基本功致胜 - 在下半场决定生死
5. 慢决策，快执行 - 深思熟虑后果断行动
6. 理性外壳，理想内核 - 表面冷静内心执着

### 八条决策启发式
1. 胜可知，而不可为
2. 核心能力决定边界
3. 以消费者价值为中心
4. 拒绝短期诱惑
5. 保持战略耐心
6. 让数据说话
7. 以终局思维做判断
8. 生存问题全力以赴

### 表达DNA
- 短句、判断句、类比隐喻
- 引用经典（《孙子兵法》《有限与无限的游戏》）
- 敢说、不撤、不道歉
- 理性外壳包裹理想内核

## 调研方法

基于**女娲造人**方法论，从四个维度深度调研：
1. **著作与系统思考** - 核心论点、自创概念、推荐书单
2. **长对话与即兴思考** - 访谈记录、立场变化、经典回答
3. **表达DNA** - 句式、词汇、幽默、争议性言论
4. **他者视角** - 竞争对手、合伙人、投资人的评价

所有信息均标注来源和可信度（五星制）。
