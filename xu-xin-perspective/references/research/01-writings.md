## 一、出版的书籍

### 1.1 本人著作：无

**重要发现：徐新未出版任何个人书籍。**

截至调研日期，徐新没有出版过任何个人著作（包括投资方法论书籍、自传、回忆录等）。这对于一位声名显赫的"风投女王"来说是不寻常的——与她同级别的投资人（如沈南鹏、张磊）均有著作问世或参与合著。

## 三、反复出现的核心投资理念（≥3次的真信念）

以下论点在多个来源中被徐新反复强调，判断为其核心信念：

### 3.1 长期主义 / 长期持有（出现次数：5+）

**信条**：伟大的公司值得长期持有，复利的力量。

**证据链**：
| 证据 | 来源 |
|------|------|
| "对好的公司，有耐心等待。坚持投少数伟大的公司" | 36氪/投资人说（2021） |
| 腾讯股票"一股未卖，有钱就加仓"，16年500倍回报 | 36氪/投资人说（2021） |
| 网易最低跌至1美元时拒绝抛售，最终8倍退出 | 新浪/商界杂志（2024） |
| "只有耐得住寂寞，才能守得住繁华" | 新浪/商界杂志（2024） |
| "耐心资本"理念反复强调 | 新浪/商界杂志（2024） |
| "买入，长期持有，复利滚动，最后获得最高收益" | 36氪/投资人说（2021） |

**智识来源**：巴菲特信徒。她自称："我读巴菲特的书和他写给股东的信就像读圣经一样，是每天必看的。"（新浪/商界杂志，2024）

### 3.2 只赌赛道第一名（出现次数：4+）

**信条**：不投行业第二，市场只会记住第一。

**证据链**：
| 证据 | 来源 |
|------|------|
| 俏物悄语失败教训："做第二没戏，市场只会记住第一" | 36氪/投资人说（2021） |
| 给京东的建议："做到行业市占率30%，领先第二1倍，才有望做到垄断" | 36氪/投资人说（2021） |
| "你一旦进入消费者的心，变成第一之后，别人是很难取代你的" | 新浪/商界杂志（2024） |
| 研究美国1920年代25个第一品牌，60-80年后21个仍是第一 | 新浪/商界杂志（2024） |

### 3.3 选人比选赛道更重要（出现次数：4+）

**信条**：投资就是投人，要找"杀手级"创业者。

**看重的创始人品质**（来自多个来源的综合）：
1. **杀手般的直觉**："看到别人看不到的东西"——如丁磊从门户转游戏自救，刘强东砸钱做自建物流
2. **快速学习能力**："小步快跑迅速迭代"，如王兴"即便很多东西都不是他先做的，但一旦做了就会打败别人变成第一"
3. **够狠，能干掉小白兔**："小白兔"指业绩不好但价值观好的员工，"一个企业能不能走得远，就看你能不能把小白兔干掉"
4. **大学就创业**："大学创业的人通常不是名利心驱使，而是天生就是创业者"，有胆量、有管理能力
5. **诚信**：刘强东即使缺钱也不愿违反保密协议的情节让她印象深刻

### 3.4 投资三步骤框架（出现次数：3+）

**框架**：行业 → 企业家 → 价格（优先级递减）

三不投：
1. 市场没需求（市场风险）
2. 有需求但生意模式不行（模式风险）
3. 人的品德有问题（人的风险）

### 3.5 专注（出现次数：3+）

"我只投消费品品牌、零售连锁、B2C"——明确的能力圈边界。

"别的投资人一年投三个，我可能三年才投一个"——重质不重量。

## 四、自创投资术语与概念

### 4.1 "杀手直觉" / Killer Instinct
- **定义**：优秀风险投资家必备的直觉能力，能"看到别人看不到的东西"
- **来源**：36氪（2021）、新浪/商界杂志（2024）
- **一手性**：一手（徐新原话）

### 4.2 "狼性企业家" / Wolf Entrepreneur
- **定义**：嗅觉灵敏、有洞察力、学习能力强、领导能力强的创业者
- **关联**：与刘强东"只有第一，没有第二"的理念相通
- **来源**：36氪（2021）

### 4.3 "小白兔" / Little White Rabbit
- **定义**：业绩不好但价值观好的员工，被徐新认为是"对企业最有危害的人"
- **来源**：36氪（2021）
- **一手性**：一手（徐新原话）

### 4.4 "Winner Pattern" / 赢家模式
- **定义**：徐新用于研究成功企业共性的分析框架。在研究此模式时看懂了内容+广告的Feed流模式
- **来源**：搜狐（2025，引用南大校友会发言）
- **一手性**：一手（徐新原话）
- **备注**：具体方法论未公开，仅在访谈中提及

### 4.5 "耐心资本" / Patience Capital
- **定义**：强调长期投资的重要性，支持创新和长期价值创造
- **来源**：新浪/商界杂志（2024）
- **备注**：2024年投资界热门概念，徐新和今日资本被视为典型代表

### 4.6 "徐新式投资" / Xu Xin-style Investment
- **定义**：强调长期主义的投资哲学，与创业者一起成长，共同创造价值
- **来源**：新浪/商界杂志（2024，商界记者创造的标签而非徐新自称）

### 4.7 "Build Business For China" / 为中国打造企业
- **定义**：今日资本的使命宣言
- **来源**：36氪（2021），据称是徐新在创立今日资本3天3夜会议中力主的定位
- **一手性**：高（今日资本官方使命）

### 4.8 "3万小时定律" / 30,000-Hour Rule
- **定义**：徐新将自己的成功归因于在投资领域积累了3万小时，"我想我可以干到80岁，还可以再来3万个小时"
- **来源**：新浪/商界杂志（2024）
- **一手性**：一手（徐新原话）

## 五、投资方法论总结

### 5.1 投资哲学核心
```
核心理念：长期主义 + 专注 + 只投第一 + 选对人
智识根基：巴菲特价值投资（她自称每天读巴菲特股东信"像读圣经一样"）
实践原则：Buy and Hold + 复合增长 + 耐心等待
```

### 5.2 投资决策流程（重构自多个来源）
1. 选赛道：找大赛道中的品类开创者
2. 选企业家：找"杀手级"创始人（直觉+学习力+狠劲）
3. 估值：最后考虑价格
4. 重仓：一次下重注，帮助被投企业建立垄断地位（市占率30%+，领先第二1倍）
5. 长期持有：不被短期波动干扰
6. 持续加仓：对看好的公司（如腾讯），有钱就加仓

### 5.3 创业者三分法（徐新原创分类）
| 类型 | 优势 | 劣势 |
|------|------|------|
| 跨国公司高管 | 系统管理、企业文化、专业性强 | 执行者而非战略者，主要是销售+管理 |
| 海归派 | 聪明、融资有优势 | 管理经验少，管人管企业经验有限 |
| 本土企业家 | 数量多、"杀手"直觉、低成本运作、善于学习 | 对董事会不懂、容易一言堂 |

### 5.4 风险管理框架
- **不投三类风险**：市场无需求 / 生意模式不行 / 人的品德有问题
- **失败教训**：俏物悄语——做第二没戏
- **认知边界意识**：错过字节跳动因为"没有看懂"内容+广告模式，错过拼多多因为"不太懂第一线人民群众怎么生活"

## 七、发现的矛盾与张力

### 7.1 投资节奏的矛盾
- **说法A**："别的投资人一年投三个，我可能三年才投一个"（新浪/商界杂志，2024）
- **说法B**：今日资本官网数据显示其投资了京东、大众点评、携程、网易、美团、知乎、赶集网、唯品会、土豆网、瓜子二手车、Boss直聘、马蜂窝、小猪短租、三只松鼠、良品铺子等大量项目
- **分析**：两种说法存在明显张力。可能原因：(1)"三年投一个"是修辞夸张；(2)徐新指的可能是亲自深度参与的项目而非所有portfolio项目

### 7.2 "只投第一"与实际行动的矛盾
- **宣称**：不做第二，只投行业第一名
- **实际**：投资知乎时，知乎当时在知识社区领域可能并非第一（C轮估值已高达3-4亿，很多人觉得贵），徐新承认"C轮错过了，一直念念不忘，于是追了进去"。投资BOSS直聘时，BOSS直聘也尚未成为行业第一。
- **分析**：可能"第一"的定义更偏向"品类开创者"或"潜在第一"，而非当下的市场排名

### 7.3 早期投资定位 vs 二级市场操作
- **宣称定位**：今日资本专注一级市场早期投资
- **实际**：徐新在二级市场建仓买入拼多多，也通过二级市场持续买入腾讯
- **分析**：她区分了"机构行为"（今日资本投一级）和"个人/灵活行为"（二级市场买入）；但拼多多的二级建仓模糊了此边界

### 7.4 "我没有看懂" vs 后续大额投资
- **关于字节跳动**：徐新说"其实是我没有看懂"错过了早期投资，但后来以近3亿美元通过老股转让入场，对应估值4800亿美元
- **分析**：她的"看懂"标准可能极高——早期投资需要理解商业模式本质，而后期投资更多基于已验证的商业模式和市场地位

### 7.5 父权叙事与独立叙事
- **叙事A**（36氪）：成功来自"人生的三次关键选择"——独立自主、个人奋斗
- **叙事B**（商界杂志）：父亲"不惜重金请家教"、"用工厂管理经验开导"、"父亲的衣架如雨点般落在她身上"——家庭资源密集介入
- **分析**：两种叙事在同一篇文章中都存在。富家女奋斗叙事的内在张力未解决。

## 九、关键结论

1. **徐新没有出版过任何书籍**，她的思想主要通过访谈、演讲、朋友圈传播，系统性不如张磊（《价值》）、沈南鹏等同行
2. **核心智识来源是巴菲特**，她明确将其奉为偶像和思想来源，每日阅读股东信
3. **"长期主义"是其最核心、最一致的信条**，在多个渠道反复出现≥5次
4. **她的方法论有原创性贡献**："杀手直觉""狼性企业家""小白兔"等概念虽非严格理论创新，但在中国VC语境中具有独特性
5. **存在言行张力**：宣称"只投第一"与实际行动有矛盾；"三年投一个"与实际portfolio数量矛盾；一级定位与二级操作有张力
6. **Winner Pattern是她最神秘的方法论**：被提及但从未详细公开，是其智识体系中最值得深入挖掘的部分
