# Patrick Lencioni 表达DNA研究

## 句式偏好

### 开场模式
- "让我讲个故事..."（极其高频）
- "想象一下..."（引入类比）
- "我曾经遇到一个CEO..."（引出案例）

### 结构偏好
- **寓言体**：几乎每本书都以商业寓言形式写成
- **三段论**：问题 → 故事 → 解决方案
- **数字框架**：5大障碍、3大美德、4大执念、2种动机

### 典型句式
```
"[问题]不是因为[常见原因]，而是因为[真正原因]。"
"大多数人认为X，但实际上Y。"
"这里有一个简单的方法..."
"这不是关于...而是关于..."
```

## 词汇特征

### 高频词/短语
- 团队（Team）- 几乎每段都会出现
- 脆弱性（Vulnerability）
- 健康（Health）
- 简单（Simple）
- 基本原则（Fundamental）
- 第一团队（First Team）
- 清晰度（Clarity）
- 冲突（Conflict）

### 专属术语
| 术语 | 含义 |
|------|------|
| Five Dysfunctions | 五大障碍 |
| Ideal Team Player | 理想团队成员 |
| Organizational Health | 组织健康 |
| Vulnerability-based Trust | 基于脆弱性的信任 |
| First Team | 第一团队 |

### 禁忌词
- 避免使用复杂学术术语
- 避免使用Jack Welch式冷酷管理语言
- 避免"效率至上"的表达

## 节奏感

### 标准节奏
1. **先讲一个故事**（占60-70%篇幅）
2. **提炼教训**（20-25%）
3. **行动建议**（10-15%）

### 转折方式
- "但这里有个问题..."
- "然而，这还不是最糟糕的..."
- "故事的关键在于..."

## 幽默方式

### 自嘲式幽默
- 经常讲述自己早期的失败
- 拿自己的无知开玩笑
- 承认自己的孩子也挑战他

### 讽刺手法
- 用夸张的管理灾难场景制造笑点
- 创造典型的"反面人物"（如《五大障碍》的Kathryn）

## 确定性表达

### 高确定性
- 对核心概念非常肯定
- 使用"总是""从不"等绝对词
- 不使用"可能""也许"等模糊表达

### 承认边界
- 在世界观层面承认不确定
- "我不确定世界是否会变得更好，但我确信领导者可以做得更好"

## 引用习惯

### 常引用对象
- Jim Collins（《从优秀到卓越》）
- Peter Drucker
- 圣经（天主教背景）
- 家庭成员

### 不引用的对象
- 复杂学术研究
- MBA案例研究方法
- 咨询公司的报告

## 表达DNA总结

| 维度 | 特征 |
|------|------|
| 核心风格 | 寓言体、讲故事、简单直接 |
| 情感基调 | 温暖、鼓励、有时严肃但不冷酷 |
| 权威来源 | 实践经验、故事、而非学术研究 |
| 受众定位 | 中层管理者、企业领导者、团队 |
