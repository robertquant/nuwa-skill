## 一、早期创业历程（1998-2004年）

### 0. 中关村柜台创业（1998年）

**时间**：1998年6月18日

**背景**：
- 1998年，24岁的刘强东辞去外企工作，揣着工作两年攒下的12000元
- 北京中关村每天诞生一百个创业者，死掉九十九个
- 刘强东在海龙市场前的"海开"二楼租下4平米柜台

**决策逻辑**（刘强东本人回忆）：
- 海淀区开发集团（海开）租金便宜：月租2200元，押一交一，其他市场要交一年
- 创立"京东多媒体"，名称取自初恋女友和自己名字各一字
- 主营"多媒体三件套"：刻录机、压缩卡（录像带转VCD）、声卡

**早期经营特点**：
- 在假货横行的中关村立下"三条铁律"：不卖假货、不偷税漏税、不行贿
- 坚持明码标价、正品行货、主动开发票
- 每月消耗30本收据本（其他柜台不开发票）
- 在柜台后面支行军床，凌晨两点还在研究产品参数
- 记住每个回头客的名字和喜好

**结果**：
- 从4平米柜台扩展到12个柜台
- 创业第三年实现1000万元营收
- 光磁产品市场占有率做到全国第一

**事后反思**：
- 2017年哈佛商学院演讲："那时候中关村有三大生存法则：不开发票、现金交易、见人下菜碟"
- "当其他柜台用翻新机赚取300%利润时，我固执地给每个客户手写发票"

**来源**：
- 搜狐网《中关村柜台后的那盏灯》（2026年）
- 腾讯新闻《中关村杂志》专访（2025年）
- 海淀区图书馆《刘强东创业故事》（2024年）
- 百家号《摆摊"前浪"刘强东》（2020年）

**可信度**：高（多家媒体报道，包含本人演讲）

### 0.1 非典转型电商（2003年）

**时间**：2003年

**背景**：
- 非典疫情爆发，中关村封闭，所有柜台被迫停业
- 大量库存积压，现金流告急

**决策逻辑**：
- 刘强东看着堆积如山的库存，决定尝试网上销售
- "当时连淘宝都没有，我们在论坛上发帖子，留QQ号，有人咨询就打字回复，谈成了再银行转账发货"
- 这是京东从线下转向线上的起点

**执行过程**：
- 2003年开始在论坛上尝试线上销售
- 2004年正式转型电商，上线"京东多媒体网"（京东商城前身）
- 彻底放弃线下柜台，全面转向线上

**结果**：
- 完成从线下代理商到线上电商的关键转型
- 2007年京东正式更名为京东商城，销售额达3.6亿元

**来源**：
- 搜狐网《致敬20年|刘强东与京东所坚守的"正道成功"》（2025年）
- 搜狐网《中关村柜台后的那盏灯》（2026年）

**可信度**：高（多家媒体报道，包含早期员工回忆）

## 二、重大战略决策

### 1. 自建物流体系（2007年）

**时间**：2007年

**背景**：
- 2006年中国社会化物流成本占GDP比重超过18%，而欧美仅6-8%，日本不到6%
- 京东早期60%的客户投诉与第三方物流服务差相关
- 投资人普遍反对，认为这是"疯狂"的重资产模式

**决策逻辑**（刘强东本人回忆）：
- "我们每建一个仓库，就要往里面砸几个亿"
- 核心判断：物流体验是电商竞争的关键，中国社会化物流成本奇高，原因在于货物搬运次数太多
- 想让用户满意，必须控制物流质量
- 2006年参观UPS总部后深受启发，决心建立自己的物流体系

**执行过程**：
- 2007年力排众议启动自建物流
- 2009年推出"211限时达"服务（早11点前下单当天送达）
- 2010年建立"亚洲一号"智能仓储概念
- 2012年濒临破产时，今日资本徐新追加投资救急
- 2017年京东物流独立运营，向社会开放
- 2021年5月港股上市，市值超2800亿港元

**结果**：
- 累计投入超1000亿建设仓储网络
- "211限时达"建立"京东=快"的心智认知
- 成为京东核心护城河
- 京东物流2024年营收达1828亿元，归母净利润61.98亿元

**事后反思**：
- 刘强东认为这是京东最正确的决策之一
- 2015年内部讲话："社会化物流成本奇高无比，我们看到了一个机会"
- 2025年世界互联网大会演讲："一干就是18年，这是京东的核心竞争力"

**来源**：
- 搜狐网《京东刘强东内部讲话：京东为什么要自建物流？》（2015年）
- 第一财经《频频放权，刘强东卸任京东物流总经理》（2019年）
- 刘强东2025年乌镇峰会演讲

**可信度**：高（多家权威媒体交叉验证，包含本人讲话）

### 2. 拒绝支付宝（2011年）

**时间**：2011年5月

**背景**：
- 2011年京东与支付宝合作中，京东80%订单为货到付款
- 支付宝在京东在线支付中仅占1-2%

**决策逻辑**：
- 刘强东公开表态："支付宝费率是其他支付方式的4倍，每年多付500-600万元"
- 费率过高是公开理由
- 更深层次考虑：
  1. 忌惮交易数据被阿里系掌握
  2. 与淘宝竞争关系
  3. 发展自家京东支付的战略需要

**执行过程**：
- 2011年5月宣布停止与支付宝合作
- 与银联合作推出新的支付工具
- 京东80%订单为货到付款，对用户体验影响有限

**结果**：
- 推动京东发展自有支付体系
- 京东支付份额远低于支付宝（2015年支付宝47.5% vs 京东支付2%）
- 2024年10月京东重新接入支付宝，时隔13年"破冰"

**事后反思**：
- 刘强东否认与淘宝竞争是主要原因
- 2024年重新接入支付宝，一定程度上承认了用户体验的考量

**来源**：
- 搜狐网《京东为什么没有支付宝支付》（2016年）
- 腾讯新闻《确定了！京东可以用支付宝付款了！》（2024年）
- 新浪财经《支付宝遭京东放弃，刘强东当初为什么要这么做？》（2017年）

**可信度**：高（多家媒体报道，当事人表态）

### 3. 进军东南亚国际化（2015-2023年）

**时间**：2015-2023年

**背景**：
- 刘强东提出"在海外再造一个京东"
- 2018年春节内部信强调国际化是核心战略

**决策逻辑**：
- 2017年设立战略部和国际业务拓展部
- 选择东南亚作为出海第一站
- 沿用国内重资产模式：自建本地仓储物流体系
- 累计投入超过100亿元

**执行过程**：
- 2015年进入印尼市场，推出JD.ID
- 2018年与泰国尚泰集团合作推出JD Central
- 自建本地物流体系：JDL Express Indonesia在印尼拥有11个仓库、250多个投递点

**结果**：
- 2023年1月宣布关闭泰国和印尼电商站点
- 转向供应链基础设施出海
- 出海十年"仍未真正突围"
- TikTok、Temu后来居上

**事后反思**：
- 刘强东承认：海外业务没有成功
- 2025年分享会新策略：不走跨境电商模式，坚持"本地电商、本地员工、本地采购、本地发货"
- "跨境电商模式只能卖便宜货，严重影响国家形象"
- 反思点：重资产模式在海外水土不服

**来源**：
- 搜狐网《复盘京东出海十年：从"再造一个京东"到"不做跨境电商"》（2025年）
- 36氪《连撤印尼和泰国站，京东国际怎么了》（2023年）
- 第一财经《京东国际业务重大调整》（2023年）

**可信度**：高（多家权威媒体交叉验证）

### 4. 业务拆分上市战略（2020-2025年）

**时间**：2020年起

**背景**：
- 京东集团2014年纳斯达克上市后
- 刘强东提出"35711"战略愿景：7家市值不低于1000亿的上市公司

**决策逻辑**：
- 内部会议强调："京东从来不是多元化公司，所有业务都基于供应链原则"
- 分拆成熟业务构建生态矩阵
- 每次上市是"资本盛宴"

**执行过程**：
- 2020年京东健康上市
- 2021年京东物流独立上市
- 2022年收购达达集团、德邦物流
- 2025年京东工业上市（第6家上市公司）
- 计划：京东产发、京东科技分拆上市

**结果**：
- 刘强东控制6家上市公司：京东、京东健康、京东物流、达达、德邦、京东工业
- 形成供应链生态矩阵
- 争议：分拆后影响业务协同

**事后反思**：
- 2024年市场讨论：分拆上市是否影响协同
- 对比阿里"1+6+N"拆分后暂缓，刘强东继续推进

**来源**：
- 搜狐网《刘强东将拿下6家上市公司》（2025年）
- 36氪《京东分拆得与失》（2024年）

### 14. 图书价格战突袭当当（2010年）

**时间**：2010年12月

**背景**：
- 当当网2010年12月在美国上市，是中国最大的图书电商平台
- 京东以3C数码起家，图书业务尚未规模化

**决策逻辑**（刘强东本人表态）：
- 微博公开宣布：京东图书业务三年内实现零毛利
- "如果哪位主管让他赚到了一分钱，那这位主管就是失职"
- 核心逻辑：图书是最高频的流量入口，把书的价格打下来，才能抢当当的高粘度用户

**执行过程**：
- 2010年12月微博宣战："全部图书4折"突袭当当核心品类
- 李国庆反击："要拿4000万元美金进行价格战"
- 刘强东成立"打苏宁指挥部"式作战模式
- 日均订单量从3000单飙升至10万单

**结果**：
- 成功破圈，图书业务快速启动
- 当当2010年上市后股价受挫
- 确立了京东"价格杀手"形象

**事后分析**：
- 用对自己影响小的品类（图书）打对手核心命脉（当当图书）
- 刘强东一贯策略：用1%的非核心品类打对手95%的核心业务
- "流血竞争"的典型打法：看谁血条更长

**来源**：
- 36氪《缺增长、创新和战斗力,不是刘强东想要的京东》（2026年）
- 搜狐网《零毛利3年!刘强东派5千"特工"突袭苏宁》（2026年）
- 每日经济新闻《电商"三国杀"》（2026年）

### 15. 与苏宁国美家电大战（2012年8月）

**时间**：2012年8月14-15日

**背景**：
- 苏宁、国美是线下家电零售巨头
- 京东试图在家电品类突破

**决策逻辑**：
- 刘强东连发6条微博开战：
  1. 声称要在全国招收5000名价格情报员派驻苏宁、国美门店
  2. 要求员工随时拿出手机比价
  3. 承诺京东所有大型家电比国美、苏宁便宜10%以上
  4. 放话："要是苏宁敢卖1元，京东就敢卖0元"
- 再次强调"零毛利"，三年内不考虑盈利

**执行过程**：
- 2012年8月14日刘强东微博宣战
- 苏宁易购下午4点回应：所有产品必然低于京东
- 国美当晚10点宣布全线商品比京东低5%
- 2012年8月15日大战正式爆发
- 成立"打苏宁指挥部"，横幅悬挂办公室

**结果**：
- 一淘网监测数据显示：全网仅4.2%商品真正降价
- 国美降价商品比例16.6%，京东大家电降价比例仅6%
- 部分大家电供应商不愿承担低价损失，促销产品无货可卖
- 消费者质疑"先涨价再降价""不当竞争"
- 发改委介入要求整改

**事后反思**：
- 刘强东原本想把价格战圈定在大家电（苏宁核心业务）
- 苏宁不愿被牵着走，将战火扩展至3C全品类
- 国美半路杀出，吃下价格战大部分红利
- 但刘强东成功留下"京东看重低价"的用户心智

**来源**：
- 网易《刘强东时隔12年再掀价格战》（2023年）
- 36氪《刘强东时隔12年再掀价格战》（2023年）
- 新浪财经《刘强东痴迷的"百亿补贴"》（2023年）
- 中国网《电商"三国杀"激荡的不只是价格》（2026年）

**可信度**：高（多家权威媒体报道，发改委介入）

### 16. 收购1号店与沃尔玛战略合作（2016年）

**时间**：2016年6月

**背景**：
- 1号店成立于2008年，曾是中国领先的快消品电商
- 2015年沃尔玛全资收购1号店，但经营状况不佳
- 京东拓展商超品类的战略需求

**决策逻辑**：
- 通过收购获得1号店品牌、网站、APP及用户资源
- 引入沃尔玛作为战略股东（获京东5%股份）
- 沃尔玛获得京东流量入口，京东获得沃尔玛供应链支持

**执行过程**：
- 2016年6月20日京东完成对沃尔玛1号店的收购
- 沃尔玛成为京东战略股东，持股约5%
- 山姆会员商店在京东平台开设官方旗舰店
- 沃尔玛门店接入京东物流和达达配送

**结果**：
- 京东获得商超品类重要资产
- 1号店品牌保持独立运营但逐渐边缘化
- 沃尔玛与京东形成深度战略合作
- 2024年沃尔玛清仓京东股份，8年合作关系结束
- 外界称："利散缘尽"

**事后反思**：
- 1号店整合效果不如预期
- 与沃尔玛合作带来的供应链价值有限
- 沃尔玛减持被解读为对中国电商市场态度变化

### 17. 收购德邦物流（2022-2026年）

**时间**：2022年启动，2026年完成退市

**背景**：
- 德邦物流成立于1996年，是大件快运行业龙头
- 2018年登陆上交所，市值一度突破300亿元
- 战略转型失败，业绩持续下滑

**决策逻辑**：
- 京东物流需要补充大件快运能力
- 解决与德邦的同业竞争问题（2022年承诺5年内解决）
- 整合仓储、干线运输资源

**执行过程**：
- 2022年京东卓风以近90亿元收购德邦控制权
- 2025年管理层全面换血：董事长胡伟辞职，京东系王振辉接任
- 2026年1月德邦宣布主动从上交所退市
- 京东以每股19元现金收购剩余股份，溢价35%
- 总耗资超42亿元

**结果**：
- 德邦成为京东物流全资子公司
- 保持独立品牌但管理层彻底京东化
- 解决同业竞争承诺
- 关联交易规模2024年达58.33亿元

**事后反思**：
- 刘强东一贯的"合规经营"理念：按承诺解决同业竞争
- 德邦创始人崔维星退出，一代快运传奇落幕
- 京东物流帝国补齐大件快运短板

**来源**：
- 新浪财经《38亿！刘强东彻底拿下德邦》（2026年）
- 网易《刘强东猛砸38亿，彻底拿下德邦》（2026年）
- 雪球《德邦卖身京东》（2026年）

**可信度**：高（港交所公告，多家财经媒体报道）

### 18. 达达集团控股整合（2022-2024年）

**时间**：2022年增持至2024年全盘接手

**背景**：
- 达达集团是中国领先的即时配送平台
- 京东需要补齐"最后一公里"配送能力
- 美团闪购日单量达1800万，入侵京东零售腹地

**决策逻辑**：
- 即时零售是未来趋势
- 京东需要与美团竞争"到店+到家"场景

### 20. 品类扩张战略（2007-2015年）

**时间**：2007年起

**背景**：
- 京东以3C数码起家，品类单一
- 需要拓宽品类提升用户留存、扩大市场规模

**决策逻辑**（刘强东思考）：
- "只有拓宽品类才能提升用户留存、扩大市场规模，同时分散对3C品类的过度依赖"
- 品类扩张分阶段推进，围绕供应链能力和用户需求
- 用高频品类（图书）带动低频品类（3C）流量

**扩张路径**：
1. **2010年图书品类**：突袭当当，建立流量入口
2. **2012年家电品类**：与苏宁国美大战，建立大家电地位
3. **2014年全品类扩张**：与腾讯战略合作，获取微信入口
4. **2015年商超品类**：收购1号店，与沃尔玛合作
5. **2016年生鲜品类**：投资七鲜超市，布局即时零售

**底层支撑**：
- 自营控品质（3C、家电坚持自营）
- 平台补生态（服装、家居引入第三方商家）
- 物流基础设施支撑全品类配送

**结果**：
- 京东从单一3C平台扩展为全品类电商平台
- 3C数码、家电仍是营收支柱
- 商超、生鲜、图书形成补充

**事后反思**：
- 今日资本徐新评价："扩品类和自建物流是刘强东做的最重要的两个决定"
- 刘强东2023年提出"战略三问"：进入新行业前必须回答行业是否有痛点、京东能否解决、能否获取长期价值

## 六、决策特征总结

### 1. 决策风格
- **力排众议**：自建物流、拒绝支付宝都曾遭投资人反对
- **长期主义**：物流投入14年亏损，最终建成护城河
- **大胆试错**：东南亚业务失败后快速调整
- **亲自介入**：关键决策亲自坐镇，不做甩手掌柜

- 一切围绕"供应链"：刘强东强调"与供应链无关的事情从来不碰"
- 成本、效率、体验：京东战略发展6字诀

### 3. 决策失误与调整
- 东南亚电商业务关闭：重资产模式水土不服
- 职业经理人实验：徐雷任CEO仅一年
- 兄弟论争议：重新定义"兄弟"引发舆论反弹

### 4. 危机应对特点
- 明州事件：先否认后低调，最终和解
